## 讲解的概念与教学方法

### 知识点1：方法定义
- **教学方法**：直接讲解 + 对比
- **核心内容**：
  - 属性写在 `__init__` 里面
  - 方法写在 `__init__` 外面
  ```python
  def bark(self):
      print("汪汪汪！")
  ```
- **关联考点**：A.1 Python 基础语法 - 方法定义

### 知识点2：方法里的 self
- **教学方法**：苏格拉底提问
- **核心内容**：
  - 方法定义时必须写 `self`
  - `self` 指向当前调用方法的对象
  - 调用时 `dahuang.bark()` → Python 自动把 `dahuang` 传给 `self`
- **关联考点**：A.1 Python 基础语法 - self

### 知识点3：self 的核心作用
- **教学方法**：代码演示 + 表格对比
- **核心内容**：
  ```python
  dahuang.bark()   # self 指向 dahuang → self.name = "大黄"
  xiaobai.bark()   # self 指向 xiaobai → self.name = "小白"
  ```
- **一句话总结**：让方法能访问"调用它的那个对象"的属性
- **关联考点**：A.1 Python 基础语法 - self
