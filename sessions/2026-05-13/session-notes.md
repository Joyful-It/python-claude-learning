## 核心背诵清单（第二天直接过）

### 一、分类四指标（必须脱口而出）

| 指标 | 公式 | 分母助记 |
|------|------|---------|
| Accuracy | (TP+TN) / All | 全对 ÷ 全量 |
| Recall（召回） | TP / (TP+**FN**) | 所有真 Yes，抓到几个（分母 = TP + 漏的） |
| Precision（精确） | TP / (TP+**FP**) | 喊 Yes 的人里，几个真 Yes（分母 = TP + 虚惊） |
| F1 | 2PR / (P+R) | 调和平均，P 和 R 都高 F1 才高 |

**防坑：** Recall 分母 = TP + FN（不是 TN），Precision 分母 = TP + FP（不是 FN）。

### 二、先验 → 正则化对应关系（默写）

```
均匀分布（常数先验）→ log p(θ)=0 → MAP = MLE（无正则化）
高斯分布（正态）    → -θ²/(2σ²)  → L2 正则 = Ridge = 趋近于 0
拉普拉斯分布       → -|θ|       → L1 正则 = Lasso = 直接变 0
```

### 三、标准化：谁需要谁不需要（核心逻辑）

| 需要（涉及"比较特征"） | 不需要（特征各算各的） |
|------------------------|----------------------|
| KNN（算欧氏距离） | 决策树/随机森林（只看阈值大小关系） |
| SVM（算间隔距离） | 朴素贝叶斯（各特征独立算概率） |
| 神经网络（梯度大小） | — |

**一句话口诀：** 两个特征凑一起算 → 标准化。每个特征自己玩 → 不用。

**反例助记：** 年龄 0-100 vs 收入 3 万-5 万。KNN 算距离时收入主导一切，决策树问"收入>5000？"只看自己阈值，不跟年龄比。

### 四、朴素贝叶斯四种

| 类型 | 数据假设 | 场景 | 助记 |
|------|---------|------|------|
| GaussianNB | 连续，正态分布 | 身高/成绩/价格 | 高斯=正态 |
| MultinomialNB | 计数（多少次） | 词频统计 | Multi=多次 |
| BernoulliNB | 0/1（有没有） | 词出现/不出现 | Bern=二选一 |
| ComplementNB | 同上，改进了不平衡 | 不平衡文本 | 互补 |

**朴素 = 所有特征独立，P(特征1|Y) × P(特征2|Y) 各算各的互不影响。**

### 五、SVM 核心

| 概念 | 人话 |
|------|------|
| 支撑向量 | 离分类线最近的那几个点，"撑"起了整条线 |
| 硬间隔 | 一个都不许错 |
| 软间隔 | 允许几个越界，C 控制罚多狠（C 大=罚狠=硬，C 小=宽容） |
| RBF 核 | 升到无限维，默认首选 |
| 不适合 | 10 万+条（太慢）、需要解释（黑盒） |

### 六、PyTorch vs sklearn（深度学习的门）

```python
# sklearn — fit() 一把梭，你不能改里面
model.fit(X, y)

# PyTorch — 亲手写每一步，想改就改
for epoch in range(100):
    y_pred = model(X)          # 前向：算预测
    loss = loss_fn(y_pred, y)  # 算错多少
    loss.backward()            # 自动求所有 w 的导数（!就这一句是自动的）
    optimizer.step()           # w = w - lr × 梯度
```

**PyTorch 只帮你干了 `loss.backward()` 一件事 — 自动求导。其余全是自己控制。**

## 间隔复习错题本

| 问题 | 错误答案 | 正确答案 |
|------|---------|---------|
| Recall 公式？ | TP/(TP+FP) | TP/(TP+FN) — 分母是"真 Yes 的总和" |
| `final_estimator=LogisticRegression` 为什么错？ | 少参数 | 少了 `()` — 传类不传实例 |
| 神经网络要不要标准化？ | 不需要（反向传播自修正） | 需要 — 特征尺度差导致梯度步长不一致 |
