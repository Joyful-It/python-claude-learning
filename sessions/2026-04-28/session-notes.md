## 学习的知识点

### 神经网络基础
| 知识点 | 内容 |
|-------|------|
| 神经网络识别数字原理 | 从像素数据学习特征 → 组合形状 → 判断最像哪个数字 |
| 神经元计算 | z = wx + b，a = σ(z) |
| Sigmoid 函数 | 把任意数字压缩到 0~1 之间 |
| 偏置 b | 调整神经元激活门槛，让模型更灵活 |
| x/w/z/a 含义 | x输入、w权重、b偏置、z原始打分、a激活输出 |
| a⁽⁰⁾/a⁽¹⁾ 符号 | 上标表示层数，下标表示该层第几个神经元 |
| w_{i,j} 含义 | 上一层第 j 个元素 → 下一层第 i 个神经元的权重 |
| 矩阵形式 | a⁽¹⁾ = σ(Wa⁽⁰⁾ + b) |
| 权重初始化 | 随机初始化（Xavier/He），不能全为0（打破对称性） |
| 反向传播 | 计算 ∂Loss/∂w，用梯度下降更新权重 |

### 机器学习 vs 深度学习
| 概念 | 说明 |
|-----|------|
| 机器学习 | 不写死规则，让模型从数据中学规律 |
| 深度学习 | 机器学习的一种，多层神经网络自动学习特征 |
| 核心区别 | 机器学习可能需要人工设计特征，深度学习自己学特征 |

### 卷积神经网络
| 概念 | 说明 |
|-----|------|
| 卷积 | 用小窗口（卷积核）在图片上滑动，检测局部特征 |
| 卷积核 | 像特征探测器，检测竖线/横线/斜线/边缘/弧线 |
| 适合图像原因 | 保留局部空间结构、参数少、能检测局部特征 |
| CNN 流程 | 输入 → 卷积层 → 激活 → 池化 → 深卷积层 → 全连接 → 输出 |
| CNN vs 全连接 | 全连接每个输入连每个神经元；卷积只看局部，同一卷积核滑动检测同特征 |
