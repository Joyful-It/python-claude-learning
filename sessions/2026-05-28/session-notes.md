## 一、残差连接（Residual Connection）

### 一句话

**把输入原样抄近道，加到输出上。**

```
正常路径：x → [层] → F(x)
残差路径：x → [层] → F(x) + x
                      ↑
                  这条横线就是残差连接
```

### 为什么需要？

网络越深越训不动——梯度从最后一层传回第一层，经过几十次乘法，小到可以忽略。

### 数学本质

```
普通层：y = F(x)        → 梯度全靠 F 这条链路
残差层：y = F(x) + x    → 梯度 = ∂F(x)/∂x + 1
                                       ↑
                                  +1 保底：梯度至少 ≥ 1
```

### 为什么系数必须是 1？

| 系数 | 正向 | 反向（梯度） |
|------|------|-------------|
| `+ 0.5x` | 信号每层衰减一半 | 梯度也衰减，深层训不动 |
| `+ 1.0x` | 信息原样保留 ✅ | 梯度至少 1 倍传回 ✅ |
| `+ 2.0x` | 信号每层放大 | 梯度也放大，容易爆炸 |

> **1 倍是唯一能保证"正向无损 + 反向梯度不衰减"的倍数。+1 在反向传播里给梯度留了高速公路。**

### 高速公路类比

```
没有残差：北京→石家庄→郑州→武汉→长沙→广州（每站必经过，堵了就全堵）
有残差：  每站旁边有条直达车道，信息可以跳过中间站直接传
         梯度也可以走直达车道传回第一层
```

## 二、LayerNorm（层归一化）

### 一句话

**把这一层的输出重新拉回"标准范围"，防止数值越传越失控。**

### 为什么需要？

```
没有归一化：
  层1输出：0.1~0.5 → 层2输出：2~10 → 层3输出：50~300 → 💥
有归一化：
  每层输出 → 归一化到均值0方差1 → 永远可控
```

### 计算过程

```
输入：x = [3, 5, 1, 7]（该层 4 个神经元输出）

μ（均值）= (3+5+1+7)/4 = 4
σ（标准差）≈ 2.24

归一化：x' = (x - 4) / 2.24 = [-0.45, 0.45, -1.34, 1.34]
                                   ↑ 均值=0，方差=1
```

### 理解核心

> LayerNorm = 每层做完后重新排队对齐，保证下一层队形不乱。

## 三、残差 + LayerNorm 的顺序

```
✅ 正确：x → 层 → 输出 + x → LayerNorm → 下一层
❌ 错误：x → 层 → LayerNorm → 输出 + x → 下一层
```

**为什么要先残差再加 Norm？** 残差加的 x 是原始输入，如果先 Norm 把输入标准化了，残差加的就不再是"原样信息"，失去了无损传输的意义。

## 四、Transformer 全流程串联

### 第一步：输入变成向量

```
输入："我 爱 机器 学习"
每个词查 Embedding 表 → 得到向量
结果：(4, 512) 矩阵，4 个词 × 512 维
```

### 第二步：位置编码

```
没有位置编码 → "我爱你" 和 "你爱我" 对模型一样
加上位置编码 → 第 1 个位置的"我" ≠ 第 3 个位置的"我"

操作：词向量 + 位置向量 = 带位置信息的词向量
```

### 第三步：Encoder（编码器）

```
输入 → ① 多头自注意力 → ② 残差+Norm → ③ FFN → ④ 残差+Norm → 下一层

自注意力核心：
  X × W_Q = Q（查询：我想找什么）
  X × W_K = K（键：我有什么标签）
  X × W_V = V（值：我的真实内容）

  注意力得分 = softmax(Q × Kᵀ / √d)
  输出 = 注意力得分 × V  →  每个词融合了全句信息

多头：并行做 8 套 QKV，每个头学会不同的"关系模式"
```

### 第四步：Decoder（解码器）

```
输入 → ① 掩码自注意力（只看左边，不能偷看后面）
     → ② 残差+Norm
     → ③ 交叉注意力（Q 来自 Decoder 自己，K,V 来自 Encoder 输出）
     → ④ 残差+Norm
     → ⑤ FFN + 残差+Norm
     → Linear + Softmax → 预测下一个词
```

### 自注意力 vs 交叉注意力

| | 自注意力（Encoder） | 交叉注意力（Decoder） |
|------|------|------|
| Q 来自 | 自己这一层 | Decoder 自己 |
| K 来自 | 自己这一层 | **Encoder 输出** |
| V 来自 | 自己这一层 | **Encoder 输出** |

> 交叉注意力：Decoder 拿着自己的问题(Q)，去 Encoder 的理解(K,V)里找答案。

## 速记卡

| 题 | 两秒回答 |
|----|---------|
| 残差连接公式？ | y = F(x) + x |
| 为什么系数是 1？ | 正向无损传输 + 反向梯度至少×1 传回 |
| +1 在反向传播中的作用？ | ∂y/∂x = ∂F/∂x + 1，给梯度留高速通道 |
| LayerNorm 做什么？ | 把输出拉回均值 0 方差 1，防止数值失控 |
| Add & Norm 顺序？ | 先残差(Add)再加 Norm，保证残差加的是原样信息 |
| 自注意力 QKV 来源？ | 全部来自同一层输入 |
| 交叉注意力 QKV 来源？ | Q 来自 Decoder，K,V 来自 Encoder |
| 多头注意力"多头"是什么？ | 并行 8 套 QKV，每头学不同关系模式 |
| Transformer 为什么能堆深？ | 残差连接撑着的，没它 12 层都训不动 |
