### 2. Python 基础：enumerate() 函数

**核心问答**：
- Q: `for i, page in enumerate(reader.pages):` 什么意思？
- A: enumerate 同时返回索引 i 和元素 page。i 从 0 开始递增，page 每次自动换成下一页。本质是 `(0, 第1元素), (1, 第2元素), ...` 的元组序列

**学员个人总结**：
> enumerate() 是 Python 最常用的内置函数之一，核心作用：遍历可迭代对象（列表、元组、字符串等）时，同时获取「元素的索引」和「元素本身」，彻底替代手动维护计数器的繁琐写法

### 5. 深度学习：RNN BPTT 梯度消失

**学员个人推导**：
> RNN 训练时使用 BPTT 进行反向传播，梯度需要跨多个时间步传播。由于链式法则会导致梯度连续相乘，当导数小于 1 时，梯度会指数级衰减，产生梯度消失问题。这样模型无法学习远距离依赖关系，因此容易遗忘序列前面的信息。

> LSTM 引入了 Cell State 和记忆门机制，通过遗忘门、输入门和输出门控制信息流动。同时 Cell State 提供了一条相对稳定的梯度传播路径，使梯度能够跨越较长时间步传播，从而缓解梯度消失问题，更好地建模长程依赖。

> BPTT = Back Propagation Through Time = 时间反向传播

**数学演示**：
> 假设每一步导数大约 0.8：
> 0.8^10 = 0.107
> 0.8^50 = 0.000014
> 0.8^100 = 0.0000002
> 几乎变成 0，前面的词完全没有梯度信号

### 6. 机器学习：Gini vs Entropy 树深

**学员个人理解**：
> "树深""树浅"不是说画出来的图大小，而是指从根节点到叶子节点

> 当前节点 100 个样本已经很纯了：
> - Gini 认为：不错了，可能直接停止或者很快找到一个切法
> - Entropy 对不纯更敏感，可能觉得还有 10 个负类混在里面，于是再长一层再长一层，树变深

> 两者都是决策树选择划分特征时衡量节点纯度的指标。Entropy 基于信息论，计算信息增益；Gini 基于样本被错误分类的概率。由于 Gini 不涉及对数运算，计算更快，因此 CART 和 sklearn 默认使用 Gini。在大多数数据集上，两者效果接近。

**sklearn 参数**：
```python
DecisionTreeClassifier(criterion="entropy")  # 也可用 criterion="log_loss"
```

### 7. 其他注入题知识点

**Chroma vs FAISS**：
> Chroma 存磁盘持久化，FAISS 纯内存极致速度（Meta 出品，百万级向量）

**Agent 死循环**：
> 调工具→不满意→再调→循环。用 ModelCallLimitMiddleware 限制最大调用轮数

**广播 vs 向量化**：
> 向量化=把循环变批量操作（C 层一次算完，比 Python 循环快几十倍）
> 广播=形状不同时 NumPy 自动对齐（如 (3,4)+(4,) → 自动扩成 (3,4)）

**三种访问级别**：
> public(self.x) / protected(self._x 约定别碰) / private(self.__x 名字改写)

**HuggingFace Pipeline NLP 任务**：
> 口诀：情分生翻摘问答 — 情感/分类/生成/翻译/摘要/QA

## 踩坑与纠正

### 晨考薄弱点（2026-06-06 晨考 20 题）

| 错误 | 正确 | 记忆口诀 |
|------|------|---------|
| Recall = TP/TN | Recall = TP/(TP+FN)，分母是所有真实正例 | Recall 看漏没漏（FN在分母），Precision 看冤不冤（FP在分母） |
| 蒸馏温度 T = "严谨性" | T>1 压平 softmax 分布，让暗知识显形 | 温度 T > 1 不是为了严谨，是为了让暗知识显形 |
| Chroma = "入库函数" | Chroma = 向量数据库（存储+检索），Embedding 模型 = 生成向量 | Chrom 存，BGE 算 |
| 测试集可以用 fit_transform | 测试集只能用 transform，fit 会数据泄露 | fit=学规矩，transform=用规矩；测试集绝不能 fit |
| 三种访问级别不知道 | public(self.x) / protected(self._x) / private(self.__x→名字改写) | 无下划线公开，单下划线别碰，双下划线碰不到 |
| 广播 vs 向量化分不清 | 向量化=批量运算，广播=形状对齐 | 广播让形状对上，向量化一把算完 |
| Agent 死循环怎么限不知道 | ModelCallLimitMiddleware 限最大调用轮数 | Agent 卡循环，ModelCallLimit 刹车 |
| Gini vs Entropy 没听过 | Gini 计算快默认用，Entropy 更敏感但慢 | Gini 快树浅，Entropy 慢树深 |
