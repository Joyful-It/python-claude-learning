## 讲解的概念与教学方法

### 知识点1：循环的引入
- **教学方法**：苏格拉底提问法
- **核心内容**：
  - 从"打印5遍你好"引出重复代码的问题
  - 引导学员思考打印100遍、10000遍的困难
  - 自然引出循环的必要性
- **关联考点**：A.1 Python 基础语法 - 循环结构

### 知识点2：if语句中的逻辑运算符
- **教学方法**：直接回答 + 表格总结
- **核心内容**：
  | 逻辑 | Python符号 |
  |------|-----------|
  | 并且 | `and` |
  | 或者 | `or` |
  | 非 | `not` |
- **关联考点**：A.1 Python 基础语法 - 逻辑运算符

### 知识点3：函数的参数
- **教学方法**：苏格拉底提问 + 课件讲解
- **核心内容**：
  1. **形参 vs 实参**
     - 形参：定义函数时的参数名
     - 实参：调用函数时传入的具体值
  2. **四种参数形式**
     - 直接参数名
     - 参数默认值（`name='python'`）
     - `*args` 接收多个位置参数（转元组）
     - `**kwargs` 接收多个关键字参数（转字典）
- **关联考点**：A.1 Python 基础语法 - 函数

### 知识点4：Git/GitHub 基础
- **教学方法**：实操演示
- **核心内容**：
  - Git 工作流程：本地创建 → git add → git commit → git push → GitHub
  - 从远程拉取：git pull
- **关联考点**：H.39 Git版本控制

### 知识点5：面向对象基础（预习）
- **教学方法**：苏格拉底提问 + 类比教学
- **核心内容**：
  1. **类与对象**
     - 学员类比：人是类（规定属性），具体某个人是对象
     - 强调：类=模板，对象=实例
  2. **属性与方法**
     - 属性 = 有什么（皮肤颜色、年龄、姓名）
     - 方法 = 能做什么（吃饭、说话、走路）
     - 学员能正确区分
  3. **__init__ 初始化方法**
     - 来源：initialize（初始化）
     - 作用：创建对象时自动执行，把属性存到对象上
  4. **self 的含义**
     - 语法上：形参，占一个位置
     - 运行时：引用/指针，存储对象的地址
     - 一句话：`self` 是一个特殊的形参，它接收的是"对象的地址"（指针）
- **关联考点**：A.1 Python 基础语法 - 面向对象

- **学员回答**：菜谱和菜；类是大类比如车，对象是细分化比如奔驰
