## 今日理解跃迁

> 从"ML 和 DL 是两套东西" → "所有模型都是同一套训练循环，区别只在三处：结构(wx+b还是树)、损失(MSE还是交叉熵)、特征谁选(你挑还是网络学)"

---

## 核心背诵链（按推导顺序，可顺序默写）

### 第一节：训练循环（所有模型的底盘）

```
① 瞎猜 w,b → ② wx+b(+激活)算预测 → ③ 损失函数看错多少 → ④ 梯度求方向 → ⑤ w=w-lr×梯度 → 回到②
```

| 工具 | 做了啥 | 怎么做到的 |
|------|--------|-----------|
| sklearn | 六步装进 fit() | `model.fit(X, y)` 一把梭 |
| PyTorch | 亲手写每一步 | `loss.backward()` 只帮你自动求导 |

**深度学习 = 机器学习 + 多隐藏层。训练六步完全一样。**

---

### 第二节：损失函数（两派）

```
预测类别（分类）：  交叉熵 = -[y×log(ŷ) + (1-y)×log(1-ŷ)]
                   → 罚"方向对不对"，猜反了往死里罚

预测数字（回归）：  MSE = (1/N)×Σ(ŷ-y)²
                   → 罚"数值离多远"，大错平方重罚

                   MAE = (1/N)×Σ|ŷ-y|
                   → 罚距离但不放大离群点
```

| | 交叉熵 | MSE | MAE |
|------|-------|-----|------|
| 罚什么 | 方向不对 | 距离平方 | 距离绝对值 |
| 用在哪 | 逻辑回归 | 线性回归 | 数据有离群点 |
| 离群点 | — | 很敏感 | 不理它 |

---

### 第三节：正则化（防 w 跑偏）

```
纯线性回归：    损失 = MSE
加 L2(Ridge)：  损失 = MSE + λ×w²    → w 趋近 0
加 L1(Lasso)：  损失 = MSE + λ×|w|  → 不重要 w 直接变 0
```

**同一形式罚不同东西：MSE 罚预测误差，L2 罚权重。数学都是²。**

| 数学形式 | 罚预测 | 罚权重 |
|---------|--------|--------|
| 平方（²） | MSE | L2（Ridge） |
| 绝对值（\|\|） | MAE | L1（Lasso） |

**正则化不是更强，是更稳。** 特征少数据多不需要，特征多或相关时能保命。

---

### 第四节：贝叶斯线（三先验 → 三正则）

```
贝叶斯公式：  p(θ|X) = p(X|θ) × p(θ) / p(X)
              后验     似然      先验    证据

MLE(频率派)： 只用 p(X|θ)，不要先验 → 数据说啥就是啥
MAP(贝叶斯)： p(X|θ)×p(θ) 都用 → 数据+信念综合
```

| 先验分布 | log p(θ) | 正则化 | 效果 |
|---------|----------|--------|------|
| 均匀分布 | 常数 | 无(=MLE) | 你啥都不信，数据全权决定 |
| 高斯(正态) | -θ²/(2σ²) | L2=Ridge | w 在 0 附近概率高 |
| 拉普拉斯 | -\|θ\| | L1=Lasso | w 在 0 处有尖角，直接踩 0 |

**先验 = 起点，后验 = 终点。训练把你从先验拽到后验。**

---

### 第五节：梯度下降（找谷底）

```
损失函数 = 海拔地图
梯度 = 斜率 = 这个方向损失在上升还是下降
梯度=0 → 地是平的 → 到谷底了 → 停下

w_new = w_old - 学习率 × 梯度
```

**梯度为 0 是"不动了"，不是"斜率为 1"。训练目标：把梯度打到 ≈ 0。**

---

### 第六节：标准化（谁需要谁不需要）

```
需要标准化（特征互相比较）：  KNN（算距离）、SVM（算间隔）、神经网络（梯度步长）
不需要（各算各的）：         决策树/随机森林（只看阈值）、朴素贝叶斯（独立乘概率）
决策依据：                  两个特征凑一起运算 → 标准化；各玩各的 → 不用
```

---

### 第七节：模型全家福（分类器）

| 家族 | 代表 | 怎么分 | 特殊需求 |
|------|------|--------|---------|
| 线性 | 逻辑回归 | wx+b + sigmoid | 要标准化 |
| 树 | 决策树 | 阈值问问题（年龄>30？） | 不需要标准化 |
| 集成 | 随机森林 | 多棵树投票 | 不需要标准化 |
| 距离 | KNN | 邻居投票（欧式/曼哈顿） | **必须**标准化 |
| 概率 | 朴素贝叶斯 | 独立概率乘 | 不需要（各特征独立） |
| 边界 | SVM | 找最大间隔线 | **必须**标准化 |

**线性回归 = 逻辑回归去掉 sigmoid。公式都是 wx+b，区别在最后一步。**

---

### 第八节：深度学习（多隐藏层）

```
隐藏层作用：  逐层学越来越抽象的特征
            像素 → 边角 → 形状 → 部件 → 整张脸
            浅层：能看懂(边角)，深层：看不懂(抽象数字)

激活函数作用：  折弯直线 → 让网络画任意形状
              每层都有，中间层 ReLU，最后 sigmoid/softmax
              没激活 → 三层叠一起 = 一层，永远画直线

损失函数：    只有最后一个，跟真实标签比差距
             中间层不打分，只折弯
```

**深度 vs 传统 ML：** 训练六步一样，区别在"特征谁选"——传统靠你手工挑，深度让隐藏层自动学。

---

### 第九节：SVM 四核函数

| 核函数 | 作用 | 使用 |
|------|------|------|
| linear | 不动，原空间画直线 | 线性可分 |
| poly | 多项式升维 | 知道大概几阶 |
| rbf | 升到无限维（默认首选） | 不知道什么形状先试它 |
| sigmoid | 模拟神经网络 | 几乎不用 |

**感知机 = SVM 去最大化间隔和核函数（1957→1990，升级版）。**

---

### 第十节：朴素贝叶斯四种

| 类型 | 数据假设 | 用在哪 |
|------|---------|--------|
| GaussianNB | 连续，正态分布 | 身高、成绩、价格 |
| MultinomialNB | 计数（出现多少次） | 词频 |
| BernoulliNB | 0/1（有没有） | 词出现/不出现 |
| ComplementNB | 同上，改进了不平衡 | 不平衡文本 |

**朴素 = 假设所有特征互相独立，P(特征1)×P(特征2) 各算各的。**

---

### 第十一节：分类四指标（必须秒答）

```
Accuracy  = (TP+TN)/All          → 全对÷全量（笼统）
Recall    = TP/(TP+FN)           → 真 Yes 抓到几个（分母含 FN 漏的）
Precision = TP/(TP+FP)           → 喊 Yes 对几个（分母含 FP 虚惊）
F1        = 2PR/(P+R)            → R 和 P 都高 F1 才高

防坑：Recall 分母 = TP+FN（不是 TN），Precision 分母 = TP+FP（不是 FN）
```
