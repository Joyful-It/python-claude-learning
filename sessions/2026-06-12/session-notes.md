## 概念与教学内容

### 1. softmax 三步骤（学员手算全对）

**Q**：softmax 怎么算？
**A**：① eˣ 求指数 → ② 求和 → ③ 每个除以和。代码 `np.exp(x) / np.exp(x).sum()`
**为什么用 e**：eˣ 导数=自己（反向传播简洁），永远为正，指数放大差距。

### 2. Transformer Attention 全公式六步拆解

**Q**：整个 Attention 公式怎么算？
**A**：六步——① X@W 出 QKV → ② scores=Q@Kᵀ → ③ ÷√dₖ → ④ softmax → ⑤ weighted V → ⑥ 输出

**Q**：Kᵀ 是什么，Q @ Kᵀ 在算什么？
**A**：Kᵀ 是 K 转置，矩阵乘法中 Q 的每个词向量去点积 Kᵀ 的每个词向量 = "我"对"所有词"的相关性分数。Q 矩阵是每人一行，各做各的，不是混在一起。

### 3. Tool vs Agent vs 多 Agent

**Q**：为什么不能每次只告诉 LLM 需要的工具
**A**：LLM 自己决定用哪个工具——在判断之前它必须看到所有工具描述

**Q**：Tool 里能调模型吗
**A**：能！但 Tool 不决定"什么时候用自己"——决策权永远在 Agent。Tool 是高级菜刀，Agent 是举刀的手

**Q**：多 Agent 到底好在哪
**A**：单 Agent+20工具 = 大海捞针易选错；多 Agent = Supervisor 从 4 个 Agent 里挑 → 子 Agent 从 3 个工具里挑 = 两次小范围精准选。类比：一个全能外卖小哥 vs 调度中心+专业骑手

### 4. 智能家教项目构思

数据流草案：学生提问 → 查画像(水平/薄弱点) → 调 RAG(课本检索) → LLM生成引导式回答(不直接给答案) → 更新画像

### 5. 项目级 Agent 第一步

**Q**：项目级 Agent 应该先做什么
**A**：画数据流图——输入→?→?→?→输出，问号填上再写代码

### 6. dₖ 和 √dₖ

**Q**：dₖ 是什么
**A**：每个注意力头 Q/K 向量的维度。d_model=512, n_heads=8 → dₖ=64

**Q**：[2,1,0] softmax 为什么不除 √dₖ
**A**：[2,1,0] 已经是除完 √dₖ 之后的温和值。除 √dₖ 在 softmax 之前，不在三步之内

**Q**：为什么 dₖ 越大点积值越大
**A**：点积是 dₖ 个数相加——加的项越多，结果波动越大。类比扔硬币。

### 7. Tensor ↔ NumPy 共享内存

`torch.from_numpy(arr)` 不复制，共享同一块内存。改一个另一个也会变。分家用 `.clone()`。
口诀："联体婴儿，一碰俩动，想分家用 clone"

### 8. Pre-LN vs Post-LN

| | Post-LN | Pre-LN |
|---|---|---|
| 顺序 | 子层→残差→Norm | Norm→子层→残差 |
| 稳定 | 需 warmup | 开箱即用 |

口诀："Pre-LN 先洗澡(Norm)再干活(Attention)再跳回(+x)"

### 9. KV Cache

自回归生成时缓存已算过的 K/V，避免每步重算全序列。O(n²)→O(n)。
口诀："不 Cache 是重写日记，用 Cache 是只写新的一页"

## 踩坑与纠正

| 错误 | 正确 | 原因 |
|------|------|------|
| AutoGPT 循环=观察→拆解→执行 | think=观察+拆解合一 → act → route | think 节点内同时读 state + 让 LLM 出牌 |
| 多态=多模态 | 多态=OOP同名不同实现，多模态=AI文本+图像+语音 | 相近中文混淆 |
| 训练循环忘记 loss 步 | "前损后步清"五步，loss 是第二步 | 长时间未复习 |
| Tensor↔NumPy 不知共享内存 | torch.from_numpy() 默认共享，改一个变两个 | 新接触的知识点 |

## 关键总结

### softmax 三部曲
```
eˣ → sum → ÷
```

### Transformer Attention 六步
```
QKV投影 → Q@Kᵀ → ÷√dₖ → softmax → ×V → 输出
```

### Tool/Agent/多Agent 一句话
```
Tool=菜刀，Agent=厨师，多Agent=大厨拆成几个小厨各管各的
```

### √dₖ 位置口诀
```
√dₖ 是 softmax 的门卫——先安检再过 softmax 大门
dₖ 大=点积飙升=softmax变独热=梯度变零=模型变傻
√dₖ 一巴掌拍回来
```
