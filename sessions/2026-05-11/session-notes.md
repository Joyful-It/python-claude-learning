## 讲解的概念与教学内容

### 1. 科学复习法（主动提取）
- 裸写 → 对答案 → 换数据 → 学新知
- 记忆留存率：重读 20% < 看代码敲 40% < 裸写 70% < 换数据 85%
- 核心原理：大脑"努力回忆但想不起来"那一刻记忆加固最强

### 2. sklearn 子模块命名规律
**记忆口诀：基础五件套 + 辅助三兄弟**
- 线(linear_model) 树(tree) 邻(neighbors) 贝(naive_bayes) SVM(svm)
- 预(preprocessing) 选(model_selection) 管(pipeline)
- 集(ensemble) → 高级货，组合弱模型变强模型

### 3. 标准化适用场景
- 必须标准化：KNN、SVM（算距离）、神经网络（梯度下降）、正则化回归
- 不需标准化：决策树、随机森林（只看大小关系）、朴素贝叶斯
- 逻辑回归：可做可不做，做了收敛更快
- 核心判断：涉及"距离计算"或"梯度下降"就先标准化

### 7. PCA vs t-SNE 详细推导
PCA：数据中心化 → 协方差矩阵 → 特征分解 → 选前 k 个 → 投影
t-SNE：高维高斯分布算 p → 低维 t 分布算 q → KL 散度最小化 → 梯度下降

核心区别：PCA 看全局方差（类在哪），t-SNE 看局部邻居（谁是邻居）

### 续：贝叶斯深度学习（同日续接）

**学员提出的问题：**
1. "为什么物理数学到最底层就会分为两类"（粒子/波、频率派/贝叶斯派、唯物/唯心）
2. "贝叶斯派的观点这个公式怎么来的"
3. "argmax 再详细一点"
4. "为什么加 log，为什么加个 log p(θ) 就是惩罚项"

**核心问答记录：**

> **问：** 为什么总是分为两类？
> **答：** 任何"描述世界"的框架都要回答两个问题——世界本身是什么（本体论）和我们对世界知道什么（认识论）。量子力学的波粒二象性最深刻：电子本身不是粒子也不是波，你用什么方式观测它就表现为什么。玻尔叫它互补原理。

> **问：** argmax 是什么意思？
> **答：** max 问"最大值是多少"，argmax 问"取最大值时自变量是谁"。类比：max 身高 = 185cm（最高多高），argmax 身高 = 张三（最高的是谁）。

> **问：** 为什么加 log？
> **答：** 两个原因。①连乘变连加，求导好算；②防止小数连乘下溢（0.1^1000 计算机当 0）。log 是单调递增，最大值位置不变。

> **问：** 为什么 log p(θ) 叫惩罚项？
> **答：** 从优化视角看，log p(θ) 很负时总分被拉低，优化器自动避开不合理区域。但从贝叶斯视角看，它不是惩罚，而是"我本来就有的信念"。

**学员今日最深理解（原话）：**

> "log p(θ) 不该说是惩罚项，而是贝叶斯的根本——就是我心里有这个想法，我想让数据往我想法靠近一点。贝叶斯太牛逼了，我现在明白之后，感觉它真牛逼。"

> "频率派只信数据，贝叶斯派我有信念，数据来修正我。先验是我心里本来就有的，不是惩罚，是我的起点。"

**学员独立推导出的知识链：**
- L2 正则化是高斯先验 → 正态分布公式取 log → 得到 -θ² 关系 → 等价于最小化 +λθ²
- L1 正则化是拉普拉斯先验 → 同样是先验取 log 推导出来的
- 先验是均匀分布（常数）→ log p(θ)=0 → MAP 退化成 MLE
- Beta 分布作为先验 → Beta(之前数据, 之前数据) → 共轭先验，后验还是 Beta
