## 2.2 变量和常量

## 2.5 Python中的注释

注释：为代码添加说明信息，注释不会被执行，只是作为说明信息

> 注释：方便理解代码逻辑的

Python语言实现注释的方式：有两种：

第一种：单行注释

语法格式：`#注释内容 `

第二种：多行注释

语法格式：

```
"""
注释的内容

"""
```

> 尽量添加注释

```python
# 注释 为代码添加说明，不会执行
# 单行注释 求和
print(1+1)
"""
多行注释
计算表达式的值
"""
print(1+1-2*3+3**2+20//3)
```

## 2.8 分支语句

> 程序的运行结构主要有3种：
>
> 1.顺序结构 程序从上往下逐行执行代码
>
> 2.分支结构（选择结构）程序有选择的执行代码
>
> 3.循环结构 程序满足条件就会重复执行

Python语言的分支结构主要有以下三种方式：

1.if语句

2.match语句

3.三目：Y if X else Z



if语句实现分支结构：

语法格式：

```
if 布尔类型表达式:
	要执行的代码块
elif 布尔类型表达式:
	要执行的代码块
……
else:
	要执行的代码块
```

> 缩进关系
>
> if语句可以单独使用，elif或else都必须配合if使用
>
> if语句内部是可以嵌套使用的

 match语句实现分支结构：

语法格式：

```
match 变量或表达式:
case 值|值:
	要执行的代码块
……
case _:
	要执行的代码块
```

> case 后面可以跟1个值，也可以跟多个值，如果多个值需要使用|进行区分，|表示 或者
>
> case _ 表示上述条件(case xx)都不满足就会执行

三目运算格式实现分支结构

语法格式：

`Y if X else Z`

> X 为布尔类型表达式，如果X为True那么就执行Y，否则就执行Z
>
> Y 和 Z 可以是一个值或表达式

```python
# 分支结构
# 变量 表示性别 1=男 2=女 3=未知
sex=input("请输入你的性别(1、2、3)：")
# if语句实现分支
if sex=='1':
    print("你的男的！")
elif sex=='2':
    print("你是女的！")
else:
    print("你的性别是未知！")
# match 语句实现分支
match sex:
    case '1':
        print("性别：男")
    case '2':
        print('性别：女')
    case _:
        print("性别：未知")
# 三目 Y if X else Z
print("性别：","男" if sex=='1' else '不是男的')
print("亲，你的性别：","男孩" if sex=='1' else ('女孩' if sex=='2' else '未知'))
```

#### 作业 5：万能数字分析器

多行注释：数字类型 + 大小 + 奇偶 + 范围 综合分析

① 判断数字是整数还是小数（type 判断）
