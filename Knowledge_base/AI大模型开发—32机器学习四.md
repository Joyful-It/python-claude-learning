# 1.SVM

## 1.1 SVM

SVM（支持向量机）一种二分类模型核心思想：在特征空间中，寻找一个最优超平面，最终实现两种分类的间隔最大化

> 1.超平面 在n维空间，超平面定义：
> $$
> w_n^Tx+b
> $$
> n=1 或2-平面 3-立体
>
> 2.支持向量：距离超平面最近的样本点，决定最终的模型
>
> 3.间隔 支持项目到超平面的距离

优势：

解决n维空间

通过核函数解决非线性的分类的问题

泛化能力相对较强

二分类：

1.w_n^Tx_i+b >=1,y_i=+1 正类

2.w_n^Tx_i+b <=-1,y_i=-1 负类

核函数:

线性核：线性

多项式核：多项式

高斯核：任意非线性

Sigmoid核：类似激活函数

# 2.贝叶斯

## 2.1 贝叶斯

贝叶斯分类器，基于贝叶斯定理实现分类

先用训练数据估计概率，再用贝叶斯公式算后验概率，最后选概率最大的类别。

> **核心逻辑**：根据以往的经验，计算“如果它是A类，那么出现这些特征的概率是多少？”
>
> 核心：
>
> - **基于概率**：它不画图，而是算概率。它想知道：“在已知特征 x的情况下，它是类别 c的概率有多大？”（即 P(c∣x)）。
> - **朴素假设**：它有一个大胆的假设——“**特征之间是互不相关的**”。比如判断一封邮件是垃圾邮件，它认为“出现了‘免费’这个词”和“出现了‘中奖’这个词”是独立的，不会互相影响。虽然现实中不一定对，但很多时候好用。
> - **连乘**：把各个特征出现的概率乘起来，就是整体的概率。
> - **一句话总结**：**它是靠“算概率”和“特征独立假设”来做判断的。**

## 2.2 贝叶斯算法推导

贝叶斯分类器：先用训练数据估计概率，再用贝叶斯公式算后验概率，最后选概率最大的类别。

## 2.3 贝叶斯实现

```python
# 贝叶斯  高斯朴素贝叶斯算法  分类
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.metrics import accuracy_score
# naive_bayes 贝叶斯  -- 概率 -- 从结果推过程--算法 先假设后验证
from sklearn.naive_bayes import GaussianNB # 高斯朴素贝叶斯

from sklearn.naive_bayes import MultinomialNB # 多项式朴素贝叶斯

print("多项式朴素贝叶斯")
model2=MultinomialNB()
model2.fit(X_train,y_train)

print("准确率：",accuracy_score(y_test,y_pred2))

# 伯努利
print("伯努利朴素贝叶斯")
model3=BernoulliNB()
model3.fit(X_train,y_train)
y_pred3=model3.predict(X_test)
print("准确率：",accuracy_score(y_test,y_pred3))

# 补集
print("补集朴素贝叶斯")
model4=ComplementNB()
model4.fit(X_train,y_train)
y_pred4=model4.predict(X_test)
print("准确率：",accuracy_score(y_test,y_pred4))
```

# 3.KNN

## 3.1 KNN

KNN K近邻算法，不需要训练模型，只需根据特征的附近的K个数据计算获取结果

KNN（K-Nearest Neighbors，K 近邻算法）是一个非常直观的“**跟着大多数走**”的分类 / 回归方法。

看一个新样本周围最近的 K 个已知样本，少数服从多数（分类）或取平均（回归）。

> 随大流

可以实现分类也可以实现回归

> 即在特征空间中：
>
> - 距离相近的点，很可能属于同一类别（分类）
> - 或具有相近的连续值（回归）
>
> 因此，对于一个新样本 x，只需要：
>
> 1. 在训练集中找到与其最近的 k个样本
> 2. 用这 k个邻居的信息来预测 x

## 3.2 KNN算法推导

需要对特征做标准化或归一化处理，因为量纲影响结果

![1778324826990](D:\class\2603\随堂笔记\第七周\AI大模型开发—32机器学习四.assets\1778324826990.png)

优缺点：

优点：思想简单、无需训练、适配非线性数据、无参数假设

缺点：预测速度慢、内存占用大、高维 / 噪声数据效果差

适用：小样本、低维数据、简单任务、快速基线模型

```python
# 基于KNN K近邻算法 实现分类
import numpy as np
from sklearn.neighbors import KNeighborsClassifier # KNN 分类
from sklearn.neighbors import KNeighborsRegressor # KNN 回归
from sklearn.datasets import make_classification
from sklearn.model_selection import train_test_split
from sklearn.metrics import accuracy_score
from sklearn.preprocessing import StandardScaler # 标准化
from sklearn.preprocessing import MinMaxScaler # 归一化
# 数据
X,y=make_classification(n_samples=100,n_features=5,n_classes=2,random_state=42)
# X=np.array([[178,0.03],[180,0.04],[190,0.05],[200,0.06],[160,0.032],[170,0.035]])
# y=np.array([0,1,1,1,0,0])
X_train, X_test, y_train, y_test = train_test_split(X,y,test_size=0.1,random_state=42)
# X 175,0.033 195 0.046
print("目标：",y_test)
# 特征的标准化 防止 量纲（特征的：数据和单位） 差异
scaler = StandardScaler()
X_train=scaler.fit_transform(X_train)
X_test=scaler.transform(X_test)

# 模型
# 验证是否近邻 距离的计算方式
# 1.曼哈顿距离 2.欧式
# metric 指定距离计算方式
model=KNeighborsClassifier(metric='euclidean')
# 没有真正的训练 只是记录了值
model.fit(X,y)

# 预测 分类 175,0.033 195 0.046
y_pred=model.predict(X_test)
print("预测：",y_pred)
# 正确率
print("正确率：",accuracy_score(y_test,y_pred))
```

基于KNN实现回归

```python
# KNN 实现 回归 平均值
from sklearn.neighbors import KNeighborsRegressor # KNN 回归
from sklearn.datasets import make_regression
from sklearn.model_selection import train_test_split
from sklearn.metrics import accuracy_score,mean_squared_error
from sklearn.preprocessing import StandardScaler # 标准化

# 数据
X,y=make_regression(n_samples=10000,n_features=20,n_informative=5,random_state=42)
X_train, X_test, y_train, y_test = train_test_split(X,y,test_size=0.1,random_state=42)
print("目标：",y_test)
sc=StandardScaler()
X_train=sc.fit_transform(X_train)
X_test=sc.transform(X_test)
# 模型
# 回归的时候一般都需要加权 采用距离的导数
model=KNeighborsRegressor(weights='distance')
model.fit(X_train,y_train)
# 预测
y_pred=model.predict(X_test)
print("预测：",y_pred)
print("MSE：",mean_squared_error(y_test,y_pred))
```
