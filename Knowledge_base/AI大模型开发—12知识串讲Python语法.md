# 1.循环

> 程序的结构只有3种：
>
> 1.顺序结构 特点：程序代码从上往下逐行进行
>
> 2.分支结构 特点：程序的代码有选择的执行，满足条件就执行
>
> 3.循环结构 特点：程序的代码会重复执行，满足条件就重复执行

Python循环结构的实现，有2种语句：

1.for语句

2.while语句

循环控制关键字：

1.continue 跳过本次循环，后续循环继续

2.break 结束本次循环，后续循环不再执行

## 1.1 for语句

for语句实现循环结构：

语法格式：

```
for 变量名 in range(起始值,结束值,步长):
	重复执行的代码块
```

> range 的作用：就是获取对应范围内的数字，其中结束值不包含
>
> [起始值,结束值) 前闭后开

for语句后面不一定跟range,只有用到数字的时候，才会使用range

还可以遍历：

字符串 获取每个字符

容器数据类型（元组、列表、集合、字典）

> 循环可以结合分支一起使用

示例代码：

```python
# 循环结构
# for语句
# 打打印1到5的数字
# 循环语句结束条件 i>=6 如果 i<6那么就一直进行循环
for i in range(1,6):
    print("数字：",i)
# for语句遍历字符串
str1='我爱河南郑州大玉米！'
for s in str1:
    print("字符：",s)
# 遍历字符串第二种方式 通过索引
for i in range(len(str1)):
    print("遍历字符串：",str1[i])
# 循环和分支可以一起使用
# 请打印100以内 可以被5整除但是不能被10整除的数字
# 第一步 100以内的数字
for num in range(1,101):
    # 第二步 5整除 但是10不能整除
    if num%5==0 and num%10!=0:
        print("想要的数字：",num)
# 循环内部还可嵌套循环
# 打印一个正方形 使用*占位
"""
****
****
****
****
行：
列：
行和列都是打印*，其中行要记得换行
"""
# 第一步 获取对应的行
for r in range(1,5): # 外层循环 获取第一行到第4行
    # 第二步 获取列
    for c in range(1,5): # 内层循环 获取第一列到第4列
        # 第三步 打印* 不换行 因为这里还是列
        print("*",end=" ")
    # 第四步 换行 列已经遍历完成
    print()

print("---直角三角形---")
# 直角三角形
"""
*
* *
* * *
* * * *
行
列：列数等于行数，第一行就只有1列，第二行有2列，第三行有3列……
"""
# 第一步 获取行
for r in range(1,6):
    # 第二步 获取列 列数=行数
    for c in range(1,r+1):
        # 第三步 打印* 不换行 因为这里还是列
        print("*", end=" ")
    # 第四步 换行 列已经遍历完成
    print()
# 打印乘法口诀表 参考上面依次类推
```



## 1.2 while语句

while语句可以实现循环结构

语法格式：

```
while 布尔类型表达式:
	重复执行的代码块
	条件的改变
```

> 务必在循环内部，实现条件的改变，否则会出现死循环（条件一直满足，重复一直进行，永不结束）

示例代码：

```python
# 循环结构
# while语句
# 输出10以内的鄂偶数
# 获取1-10的数字
num=1
while num<=10: # while循环
    if num%2==0: # 验证是否为偶数
        print("偶数：",num)
    num+=1 # 自增1 挨个获取1-10的数字
# while嵌套使用
# while 打印长方形 矩形 3*6
# 1.获取行
r=1
while r<=3:
    # 2.获取对应的列
    c = 1
    while c<=6:
        # 3 打印内容
        print("*",end=" ")
        c+=1 # 条件改变
    # 4.换行
    print()
    r+=1 # 条件改变
# 控制循环的关键字
# continue 跳过本次循环
# 1-10数字，3的倍数不打印
num1=0
while num1<=10:
    num1 += 1
    if num1%3==0:
        continue # 跳过本次循环
    print("跳3：",num1)

# break 结束循环
for i in range(1,11):
    if i%3==0:
        break # 结束循环
    print("遇到3就结束：",i)
```

# 2.函数

## 2.1 函数

函数：实现特定功能的代码块

作用：实现逻辑的封装，方便复用

语法格式：

```
# 定义函数的格式
def 函数名(参数名,……):
	实现特定功能的代码块
	return 返回值
# 调用函数的格式
变量名=函数名(参数的值,……)
# 如果调用的时候，只写值，那么就是按照函数定义的时候的参数的顺序进行赋值
# 如果想要任意指定对应的参数的值，那么需要写：参数名=值

```

> 函数 可以没有参数，也可以有1个或多个参数
>
> 函数 可以有返回值，也可以没有返回值
>
> 函数名的命名规范：采用蛇形命名法，单词都是小写的，每个单词通过 下划线 连接
>
> 什么时候需要函数：如果某个功能需要反复使用，那么就需要把这个功能封装到函数中，后续都是通过函数名来使用

示例：

```python
# 函数
# 判断数字是否为偶数
# 定义函数
# 形参： num
def is_oushu(num):
    if num%2==0:
        return True
    else:
        return False
# 调用函数
# 实参：5
print("是否为偶数：",is_oushu(5))
print("是否为偶数：",is_oushu(10))
print("是否为偶数：",is_oushu(11))
```

## 2.2 函数核心

**参数**

1.普通参数

2.参数的默认值

3.可变参数-元组 *

4.可变参数-字典 **

示例：

```python
# 函数的参数
# 函数的定义
# 普通参数
def fun1(num1,num2):
    return num1+num2
# 参数给默认值 不传就要使用默认值
def fun2(num1,num2=10):
    return num1*num2
# 可变参数：可以传递0或多个值 内部是作为 元组使用的
def fun3(num1,*nums):
    s=0
    for i in nums:
       s+=num1*i
    return s
# 可变参数：可以传0或多个值（键值对） 内部是作为 字典使用的
def fun4(name,**scores):
    for k,v in scores.items():
        c=''
        match k:
            case 'c1': c='语文'
            case 'c2': c='数学'
            case 'c3': c='英语'
        print(name,"成绩信息：",c,v)
# 函数的调用
print("普通参数：",fun1(1,1))
print("默认值的参数：",fun2(100))
print("可变参数-元组：",fun3(2,10,20,30,22))
print("可变参数-字典：")
fun4("张三",c1=60,c2=88,c3=99)
```

返回值

​	使用return进行值的返回

> 1.要么有返回值（无论什么条件都必须有返回值），要么没有返回值
>
> 2.返回值的数据类型，必须一致

特殊函数

​	递归函数：函数内部出现了自己调用自己的情况（请避免死递归，没有出口或出口永远也满足不了）

​	Lambda函数：函数的简化的格式，语法格式：lambda 参数名:表达式

函数是可以赋值给某变量

```python
# 特殊的函数
# 递归函数 函数内部出现了自己调用自己
# 斐波那契数列 1 1 2 3 5 8 13 ……
# 定义函数，获取指定位数的斐波那契数列的值
# 发现规律：前两项都是1，第三项开始，每一项=前一项+前前一项
def get_fbri(num):
    if num<3:
        return 1
    else:
        # 从第三位开始 =前一项（num-1）斐波那契数列值+前前一项（num-2）斐波那契数列值
        return get_fbri(num-1) + get_fbri(num-2)
# 调用函数
print("第6位斐波那契数列的值：",get_fbri(6))
print("第11位斐波那契数列的值：",get_fbri(11))

# Lambda函数 简化 简单函数 lambda函数 一行函数写完
def fun1(num1,num2):
    # return num1+num2
    if num1%2==0:
        return num1+num2
    else:
        return num1-num2
print("普通函数求和：",fun1(1,11))
# 使用：Lambda函数进行替代
lf=lambda num1,num2:num1+num2 if num1%2==0 else num1-num2
print("Lambdas函数求和：",lf(1,11))
```

4.循环 让用户猜单词

5.循环内部：验证每次猜的字母是否匹配

​	匹配：替换_ 

​	不匹配：记录错误次数

​	都需要记录下来已经猜过字母

6.循环结束

​	1.猜对啦

​	2.猜错次数达到上限

示例：

```python
# 实现猜单词小游戏
import random
# 1.准备词库
words=["python","car","phone","apple","llm","hello"]
print("***猜单词小游戏***")
# 2.随机选择一个单词
word=words[random.randint(0,len(words))]
show_word="_" * len(word)
#3.准备一些变量
# 错误的次数
error_num=0
# 猜过的字母
guess_letters=[]
# 猜的第几个字母
i=0
# 常量 最大的错误次数
MAX_ERROR=6
# 已经猜对的内容
str1=''
# 提示
print(f"亲，请注意你一共可以猜错：{MAX_ERROR}次！")
# 记录结果
r=False
# 4.循环 猜字母
while error_num<MAX_ERROR:
    print(f"要猜的单词：{show_word}")
    if len(guess_letters)>0:
        print("你猜过的字母："+" ".join(guess_letters))
    # 5. 获取用户猜的字母
    letter=input(f"请输入第{i+1}个字母：")
    # 记录猜的单词
    guess_letters.append(letter)
    # 验证是否猜对
    if letter==word[i]:
        # 猜对啦
        # 替换对应位置上的下划线
        str1+=letter
        show_word=str1+"_"*(len(word)-len(str1))
        # 验证是否猜完
        if i==len(word)-1:
            # 游戏结束 猜对啦
            r=True
            break
        else:
            # 更改索引
            i+=1
    else:
        # 猜错了
        error_num+=1
        print("错误次数+1，当前错误次数：",error_num)

print(f"恭喜你，猜对啦，胜利✌️" if r else "很遗憾，你不行！菜鸡！")
```
