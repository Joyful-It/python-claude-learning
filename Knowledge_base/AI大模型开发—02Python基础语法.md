## 1.1 数据类型

数据类型：数据的类型

12--数据类型 整型

12.34 --数据类型 浮点数

数字类型（整型-int,浮点型-float）：特点 ：阿拉伯数字 示例：1 345 23.34 

字符串类型(str)：特点：带有单引号或双引号 示例： 'abc' "www"

复数类型（complex）：复数是有实数和虚数组成，特点：实数(数字)+虚数(j) 示例： 1+2j

布尔类型(bool)：表示真或假，特点：值只有2种，要么True要么False 示例：True False

容器类数据类型：元组、列表、集合、字典

面向对象-类

可以使用函数：type(对应的值) 获取值的数据类型



示例：

```python
# 数据类型 标记数据的类型
# type函数 获取数据的数据类型
print("数据类型：")
# 整数---整型 int
print(type(12))
# 小数 --- 浮点型 float
print(type(1.2))
# 布尔类型 bool
print(type(True))
# 字符串类型 str
print(type('abc'))
# 复数=实数+虚数 complex
print(type(1+3j))
```

## 1.2 变量和常量

## 1.5 类型转换

不同数据类型的数据，可以实现转换

### 1.5.1 隐式转换

隐式转换（自动转换）：某些数据类型的数据可以自动的转换为其他的数据类型

比如：整型自动转换为浮点型、布尔类型参与运算自动转换为整型

> 系统会根据实际情况进行自动的转换处理

示例：

```python
# 数字类型：整型和浮点型
num1=10
num2=12.3
# 求和的结果类型
num3=num1+num2
print(num3)
print(type(num3))

# 布尔类型
bool1=False
num4=12
# 布尔类型可以隐式转换为整型 True=1,False=0
num5=bool1+num4
print(num5)
print(type(num5))
```

### 1.5.2 显式转换

显式转换（强制转换）：人为的把某种数据类型的数据转换为其他的数据类型的数据

涉及到以下4个函数：

1.转换为整型：int(xxx)

2.转换为浮点型：float(xxx)

3.转换为字符串：str(xxx)

4.转换为布尔类型：bool(xxx)

示例：

```python
# 浮点型转换为整型 显式转换-强制、手动
tem=33.9
print("原始类型：",type(tem))
# 转换为整型 显式转换
num=int(tem)
print("转换后的值：",num)
print("转换后的数据类型：",type(num))

num2=-10.2
# 把整型或浮点型 转换为布尔类型：0为False,其余为True
bool1=bool(num2)
print("类型：",type(bool1))
print("转换的值：",bool1)

num3=3
# 把整型转换为字符串类型
str1=str(num3)
# 报错，+ 算术运算符 只能连接2个数字类型
# print(num3+str1)
print(str1)
print("类型：",type(str1))

# 浮点型
str2='3.14'
# 下面这行代码会报错 类型的原因
# print(str2+11)
# 把字符串类型 转换为浮点型
float1=float(str2)
# 算术运算符
print(float1+11)

```

### 1.6.2 输出

输出：把内容输出到控制台

print函数

输出的内容可以是1个也可以多个

print(输出的内容,……)

格式化输出，在输出的内容中直接使用变量

print(f"xxx{变量名}")

设置结尾的符号，默认为换行，通过end='xx'

print(xx,xx,end='xxx')

使用%d或%s进行格式化输出

print("xx%d,%s"%(第一个数字,第二个字符串))

示例：

```python
# 输出
print("hello")
# 输出多个内容
print("hello","我是豆包",1528)
# 格式化输出
name="邢朋辉"
age=18
print(f"我叫：{name}，今年：{age}岁")
# 输出 使用end参数设置结尾的格式，默认结尾为换行
print(11,end='--')
print(1123)
print("qw")
```
