## 1.5 核心术语

| 术语                   | 专业定义                             | 通俗理解                    |
| ---------------------- | ------------------------------------ | --------------------------- |
| 特征（Features）       | 模型的输入变量，用于描述样本的属性   | 做题的 “已知条件”           |
| 标签（Labels）         | 监督学习的目标变量，模型要预测的结果 | 做题的 “标准答案”           |
| 样本（Samples）        | 数据集中的一条独立观测数据           | 一道题目 / 一个实例         |
| 模型（Model）          | 从数据中学到的数学映射函数           | 学会的解题方法              |
| 训练（Training）       | 模型迭代优化参数的学习过程           | 刷题练习                    |
| 预测（Prediction）     | 用训练好的模型对新数据推断           | 做新题                      |
| 泛化（Generalization） | 模型对未见过数据的预测能力           | 举一反三能力                |
| 过拟合（Overfitting）  | 模型记住训练数据噪声，泛化差         | 死记硬背，不会变通          |
| 欠拟合（Underfitting） | 模型太简单，无法捕捉数据规律         | 没学会，连基础题都做错      |
| 偏差（Bias）           | 模型预测值与真实值的整体偏离程度     | 系统性错误，总不准          |
| 方差（Variance）       | 模型对数据波动的敏感程度             | 稳定性差，换数据就崩        |
| 超参数                 | 训练前手动设置的参数（非学习得到）   | 学习方法 / 策略（如学习率） |

## 3.2 线性回归

线性回归：是线性模型中最重要的，也是最基础的算法实现

线性回归假设**输出是特征的线性组合**

内部又分为：一元和多元

对应的是线性方程

线性回归的评测指标如下所示：

| 指标         | 英文 | 专业解释                 | 通俗解释               |
| ------------ | ---- | ------------------------ | ---------------------- |
| 均方误差     | MSE  | 误差平方的均值           | 整体误差大小           |
| 均方根误差   | RMSE | MSE 开根号               | 误差的实际单位，更直观 |
| 平均绝对误差 | MAE  | 误差绝对值的均值         | 平均错多少             |
| 决定系数     | R²   | 模型可解释数据方差的比例 | 模型拟合程度（0~1）    |

对应sklean库的LinearRegression

数学公式：
$$
y = \beta_0 + \beta_1 x_1 + ... + \beta_n x_n + \epsilon
$$
实现步骤：

1.准备训练数据（数据和对应的结果数据）

2.看情况选择是否需要对应的线性方程

3.模型训练 fit

4.获取对应的：斜率、截距、MSE、RMSE、MAE、R的平方

5.预测 给一个测试数据，看一下预测的结果和实际结果

> 起码你得知道这个线性回归 是好还是坏

示例代码：

```python
import math
import numpy as np
from sklearn.linear_model import LinearRegression
from sklearn.metrics import mean_squared_error,mean_absolute_error,r2_score

# 线性回归的评价指标：
# 1.MSE
# 2.RMSE
# 3.MAE
# 4.R^2
# 线性方程 一般是直线 斜率 和 截距
# 线性方程 数据X 二维数组 数据y 目标值
# 1.数据X 训练数据
X=np.array([[10,20],[3,5],[5,9],[2,3]])
print("训练数据：",X)
# 2.线性方程
# y=2*x0+4*x1+6
y=np.dot(X,np.array([2,4]))+6
print("目标值：",y)
# 3.训练 线性回归模型
reg=LinearRegression().fit(X,y)
# 4.预测
# 要预测的X的数据
X2=np.array([[1,1],[2,2]])
# predict 获取线性回归的预测值
y2=reg.predict(X2)
print("预测的结果值：",y2)
print("计算目标值：",np.dot(X2,np.array([2,4]))+6)
# 线性回归 斜率和截距
print("斜率：",reg.coef_)
print("截距：",reg.intercept_)
# 线性回归的评测的指标
y1=reg.predict(X)
print("预测X的目标值：",y1)
# MSE 误差的平分和的均分
mse=mean_squared_error(y1,y)
# sqrt 平分根
rmse=math.sqrt(mse)
# MAE 误差的绝对值和的均分
mae=mean_absolute_error(y1,y)
# 评分 拟合度 R^2
r=r2_score(y1,y)
print(mse,rmse,mae)
# 3个误差 线性回归 都是0
print(f"MSE均方差：{mse:.2f}")
print(f"RMSE均方根差：{rmse:.2f}")
print(f"MAE均方绝对差：{mae:.2f}")
print("R^2拟合度：",r)
```

## 3.3 综合练习

练习1：

x = [1, 2, 3, 4, 5, 6]

y = [3, 5, 7, 9, 11, 13]

要求：

1. 训练一元线性回归
2. 输出斜率、截距
3. 计算 MSE、RMSE、MAE、\(R^2\)
4. 预测 x=7、x=8 的 y 值

示例代码：

```python
import math

import numpy as np
from sklearn.linear_model import LinearRegression
from sklearn.metrics import mean_squared_error,mean_absolute_error,r2_score

# 数据
X1=np.array([[1], [2], [3], [4], [5], [6]])
y1=np.array([3, 5, 7, 9, 11, 13])
# 训练 线性回归
reg=LinearRegression().fit(X1,y1)
# 斜率和截距
print("斜率：",reg.coef_)
print(f"截距：{reg.intercept_:.2f}")
# 评价指标
# 获取预测值
y2=reg.predict(np.array([[2],[3],[4]]))
# MSE
mse=mean_squared_error(y2,np.array([5,7,9]))
print(f'MSE均方方差：{mse:.2f}')
# RMSE
rmse=math.sqrt(mse)
print(f"RMSE均方根方差：{rmse:.2f}")
# MAE
mae=mean_absolute_error(y2,np.array([5,7,9]))
print(f"MAE均方绝对方差：{mae:.2f}")
# R^2
r2=r2_score(y2,np.array([5,7,9]))
print(f"R^2决定系数：{r2:.2f}")
print("预测：",reg.predict(np.array([[7],[8]])))
```
