## 1.2.函数是什么

函数：实现特定功能的代码块

目的：简化代码，复用性，封装逻辑代码，可维护性

语法格式：

```python
# 定义函数
def 函数名(参数名,……):
	实现特定功能的代码
    return 返回值
# 调用函数
	函数名(值,……)
```

示例：

```python
# 函数 实现特定功能的代码块
# 定义函数 实现2个数字的求和
def sum(num1,num2):
    return num1+num2
# 调用函数
print("求和：",sum(1,1))
print("求和：",sum(10,10))

```

            r = False
            break
    return r

## 1.3.函数的参数

参数：形式参数-形参和实际参数-实参

定义函数的时候：形参

调用函数的时候：实参



定义函数的参数：

1.直接参数名

2.参数名=值

​	在定义函数的时候，可以给参数设置默认值，在调用函数的时候，如果不传递参数那么就使用默认值

参考示例：

```python
# 参数的默认值
def show(name='python'):
    print("欢迎 ",name)

show() # 不传 就使用默认值
```

3.在参数前面加*：接收多个值，会被转为元组

```python
# *参数名 只能作为最后一个，可以接收0或多个值，转换为元组
def show3(name,*scores):
    # scores 作为元组
    print(name)
    for score in scores:
        print(score)

# 调用函数
show3("李四")
show3("李四",11,22,33,44,55)
```

4.在参数的前面加**：接收多个值，会转为字典

```python
# **参数名 只能最后面  多个对应的参数转换为 字典
def show4(name,**courses):
    print(name)
    for c in courses.keys():
        print(c,"的成绩：",courses[c])

# 调用函数 ** 接收0或多个键值对
show4("赵武")
# 传递多个参数，键名=值 多个之间 逗号隔开
show4("王麻子",s1=88,s2=0,s3=88)
```

## 1.4.函数的返回值

函数可以设置返回值，需要使用关键字 return

return作用：1.返回值 2.结束函数

return的用法：

1.只写return 作用：结束函数

2.return 一个返回值 作用：结束函数并返回值

3.return 多个值 作用：返回多个值

示例代码：

```python
# 函数的返回值
# return 返回值
def fun1(num):
    if num % 2 == 0:
        return True
    else:
        return False
r1=fun1(10)
print(r1)
# 只用return 没有具体的返回值 或不写return 函数返回：None
def fun2(num):
    print(num)
    return
r2=fun2(10)
print(r2)
# return 返回多个值 自动转换为元组
def fun3(num):
    return num * 2,num*5,num*7
r3=fun3(10)
print(type(r3))
```

## 1.6.几种函数形式

函数的实现的形式有以下几种：

1.普通函数

def 函数名(参数名,……):

​	函数的内容

​	return 返回值

2.嵌套函数

可以在函数内部在定义函数

示例：

```python
# 嵌套函数
# 处理字符串
def fun(s):
    #  替换空格
    s1=s.replace(" ","")
    print("---",s1)
    # 转换为大写
    return s1.upper()

print(fun("hello AI LLM"))

# 嵌套函数 在函数的内部在定义函数
def fun1(s):
    # 内部定义子函数
    def child1(s):
        return s.replace(" ","")
    # 调用子函数
    return child1(s).upper()

print(fun1("i love LLM"))
```

3.递归函数

函数内部自己调用自己

> 防止死递归，没有出口

```python

# 递归函数：函数内部出现了自己调用自己
def fun2(num):
    if num>10:
        return num
    else:
        # 调用函数
        return fun2(num+2)

print(fun2(3))

# 斐波那契数列  1 1 2 3 5 8 递归函数
def fun3(num):
    if num>2:
        return fun3(num-1)+fun3(num-2)
    else:
        return 1
print(fun3(6))
```

4.lambda函数

一行函数简化写法

作用：可以作为参数进行传递

语法格式：lambda 参数名:表达式

```python
# lambda函数 语法格式： lambda 参数名:表达式
f=lambda x:x*2
print(f(3))
list1=[11,3,5,77,80]
f2=lambda x:x>10
for i in list1:
    print(f2(i))
```

```python
# 写出内容到文件中
# with 可以不用手动关闭文件
# open 打开文件，参数：1.文件路径和文件名 2.模式 r读取 w写出
with open("test1.txt","w") as f:
    # write 写出
    f.write("Hello,World!\n")
    f.write("Hello,World!\n")
    f.write("Hello,World!\n")
    f.write("Hello,World!\n")

# 把20（可变）以内的偶数写出到文件中，并在末尾得出偶数的和

# 定义函数 实现功能
def fun1(file_name,max_num):
    # 定义变量
    s=0
    # 打开文件
    with open(file_name,"w",-1,"utf-8") as f:
        for i in range(1,max_num+1):
            if i%2==0:
                f.write("偶数："+str(i)+"\n")
                s+=i

        f.write("和="+str(s))

# 调用函数
fun1("test101.txt",20)
fun1("test102.txt",30)
fun1("test103.txt",50)
```
