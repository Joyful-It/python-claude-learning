# 1.分支

分支：条件判断 ，如果 否则

有选择的进行处理

> 程序的执行结构：
>
> 1.顺序结构：程序从上往下依次执行
>
> 2.分支结构：满足条件就执行，不满足就不执行
>
> 3.循环结构：满足条件就重复执行，不满足就结束重复

Python语言实现分支结构有三种方式：

1.if语句

2.三目运算符

3.match语句

## 1.1 if

if语句实现分支结构（选择结构）

if语句可以单独使用，也可以配合else一起使用，还可以配合多条件elif使用

语法规则：

```python
if 布尔类型表达式:
    满足条件，要执行的代码
    ……
elif 布尔类型表达式:
    满足条件，要执行的代码
    ……
……
else:
    上述条件都不满足，执行的代码
```

> if语句可以单独使用
>
> if 和 elif 多条件满足一个，if可以跟多个elif
>
> else后面不需要写表达式

示例：

```python
# 分支结构 选择结构
# if语句
score=59
# 单独使用if
if score>=60:
    print("及格万岁！")
# if和else 结合
if score>=60:
    print("及格了！")
else:
    print("不及格更得努力！")

# if和elsif 结合实现多条件
if score>=90:
    print("优秀")
elif score>=80:
    print("良")
elif score>=60:
    print("合格")
else:
    print("不合格")
```

## 1.2 三目运算符

三目运算符也是实现分支的一种方式

> 在其他主流语言中，三目运算符的格式如下所示：
>
> X?Y:Z
>
> X 布尔类型的，如果为True 那么结果为Y，否则为Z
>
> 但是Python语言中的三目运算符的实现方式不一样

Python中实现三目运算符的语法格式如下所示：

```
结果1 if 条件 else 结果2

Y if X else Z
```

> X 布尔类型 如果True那么就执行Y,否则就执行Z
>
> 三目运算符实现分支 主要是简化 if esle 

示例：

```python
# 三目运算符实现分支结构
isLogin=True
# 三目 简化版的if else
# Y if X else Z  其中 X为布尔类型
str='登录成功' if isLogin else '登录失败'
print(str)

# 上述代码等同于下面
if isLogin:
    print("登录成功")
else:
    print("登录失败")
```

## 1.3 match case

Python3.10+版本之后，Python语言新增一种分支结构实现方式

match语句

语法格式：

```
match 变量或表达式(必须有值):
	case 值:
		满足条件，执行的代码
	……
	case _:
		上述都不满足，执行的代码
```

> 注意：
>
> 1.case 后面只能跟具体的值
>
> 2.| case后面 多个值 表示：或者 比如：case 1|2|3 只要match匹配到1 或2 或3 都是执行这个
>
> 3.case _ 表示是否则，上述条件都不满足执行这个

match语句 代替的是if elif …… else

示例：

```python
# 有一个变量 表示 一年四季
num=1
# 先用 if elif else 实现
if num==1:
    print("春天")
elif num==2:
    print("夏天")
elif num==3:
    print("秋天")
elif num==4:
    print("冬天")
# 再用match实现
# match 实现多条件的判断执行
match num:
    case 1:
        print("春天")
    case 2:
        print("夏天")
    case 3:
        print("秋天")
    case 4:
        print("冬天")
# 实现一个成绩的分类 怎么match  成绩是输入的
"""
>=90 A 90-100 
>=70 B 7 8
>=60 C
<60 D
"""
score=int(input("请输入你的成绩："))
# // 向下取整
match score//10:
    case 10|9: # | 表示 或者
        print("A")
    case 8|7:
        print("B")
    case 6:
        print("C")
    case _: # case _ 表示 上述都不满足，执行 等同于 else
        print("D")

```

## 2.1 while循环

while语句，实现循环结构

```
while 布尔类型表达式:
	重复执行的代码
	条件改变，防止死循环
```

```python

    print(num)
    num+=1 #改变条件
# 循环 重要的事情讲三遍
num1=1
while num1<4:
    print("重要的事情讲三遍")
    num1+=1
```

1.如何使用while语句打印乘法口诀表？

```python
# 实现一个4*6的长方形 4行6列
i=1 # 定义初始行
while i<5: # 循环获取行
    j=1 # 定义初始列
    while j<7: # 循环获取列
        print("*",end=' ') # 每列输出的内容 end参数避免换行
        j+=1 # 列数的变化
    print() # 换行
    i+=1 # 行数的变化

print("--思考题--")

# 乘法口诀表
"""
1*1=1 
1*2=2 2*2=4
1*3=3 2*3=6 3*3=9
……
1*9=9 

每一行的列数=对应的行数
"""
i=1
while i<10: # 外层循环 控制行数
    j=1
    while j<=i: # 内层循环 控制列数
        s=j*i
        print(f"{j} * {i}= {s} ",end="    ")
        j+=1
    print()
    i+=1
```

## 2.2 for循环

for语句实现循环结构

语法结构：

```
for 变量名 in range(起始值,结束值):
	重复执行的内容
```

> for语句一般可以通过range进行循环，还可以遍历 字符串，容器数据类型

示例：

```python
# for语句实现循环
for num in range(1,10): # [1,10) >=1 <10的范围
    print(num)

# 使用for语句输出10以内的奇数
# 1.获取1-10的数字
for num in range(1,11):
    # 2.是否奇数
    if num%2==1:
        print("奇数：",num)

# for循环遍历字符串
for s in 'Hello':
    print(s)
```

> range函数
>
> 第一个参数：起始值，包含
>
> 第二个参数：结束值，不包含
>
> 第三个参数：起始值每次改变的数字

实现10-1从大到小的输出：

```python
# 从大到小输出 10-1
for i in range(10,0,-1):
    print(i)
```
