# 1.循环进阶

## 1.1 循环控制关键字

continue:跳过本次循环

break:结束循环（跳出循环）

示例：

```python
# 循环控制关键字
# 1-10 遇到3的倍数就跳过，其余就正常输出
for n in range(1,11):
    if n%3!=0:
        print("第一种实现：",n)
# continue 跳过本次循环
for n in range(1,11):
    if n%3==0:
        continue # 跳过本次循环
    print("第二种方式：",n)
# break 结束循环
for n in range(1,11):
    if n%3==0:
        break # 结束循环，跳出循环，退出循环
    print("----",n)
```

**题目2：**输入行数，用循环打印对应行数的**倒直角三角形**

```
* * * *
  * * *
    * *
      *
```

> 分析： 找规律
>
> 行和列和*和空格
>
> 规律1的形式：
>
> 空格：行数-1
>
> *：总行数-当前行数+1
>
> 规律2的形式：
>
> 空格和*与行数和列数的关系：
>
> 空格：行数> 列数
>
> *行数<=列数

参考代码实现：

```python
# 倒直角三角形
# 第一种实现 空格的数量 和星的数量 与行的关系
# 1.1 行
row1=int(input("请输入大小："))
# 1.1 获取行
for r in range(1,row1+1):
    # 1.2 打印空格
    for k in range(1,r):
        print(" ",end=" ")
    # 1.3 打印*
    for x in range(1,row1-r+2):
        print("*",end=" ")
    # 1.4 换行
    print()

print("---第二种实现--")
row2=int(input("请输入大小："))
# 2.1 行
for r in range(1,row2+1):
    # 2.2 列
    for c in range(1,row2+1):
        # 2.3 打印空格
        if r>c:
            print(" ",end=" ")
        # 2.4 打印*
        else:
            print("*",end=" ")
    # 2.5 换行
    print()
```

# 2.字符串

## 2.1 字符串常用操作

字符串类型：str，多个字符的组合

值：需要使用单引号或双引号或三引号

字符串类型的操作

1.索引

> 整型，可以获取指定位置的字符，从0开始的
>
> 字符串变量名[索引值]

2.切片

> 获取字符串中指定索引范围的子字符串
>
> 格式：[起始索引:结束索引:步长]

3.获取字符串的长度 len

4.去除首位空格 strip

5.替换指定的字符 replace

6.拼接字符串 +

7.切割字符串 split

8.in 或 not in 是否在字符串中

9.转换为大写 upper ，转换为小写 lower

10.查找对应的索引：find

示例：

```python
# 字符串类型
str1='abcdedfrtg'
# 索引 0开始 获取对应位置的字符
print("索引：",str1[1])
# 切片 获取指定索引范围内的子串 参数的意义：1.起始索引 2.结束索引(不包含) 3.步长(改变大小，默认1)
print("字符串切片：",str1[1:7:2])
# 从0开始到2结束
print("切片：",str1[:3])
# 从0开始，每隔1个取字符
print("切片：",str1[::2])
# 从指定索引开始到末尾
print("切片：",str1[2:])
# 从后往前取
print("切片：",str1[6:1:-2])
# 获取字符串的长度 len
print("长度：",len(str1))
# 去除首位的空格 strip
str2="123456  "
print(str2)
print("去除：",str2.strip())
# 替换字符串
print("替换字符串：",str2.replace("234","www"))
# 迭代字符串 join
print("拼接字符串：",str2.join("-0="))
# 拼接 使用+
print("拼接字符串：",str2+"pppp")
str3='wERt'
print("转换大写：",str3.upper(),"转换为小写：",str3.lower())
str4="zhangsan@lisi@zhaowu"
# 切割字符串 变为 列表
print("切割：",str4.split("@"))
# 是否存在
print("是否存在：", "lisi" in str4)
# 查找某个字符或子串 在字符串中的索引 
print("查找：",str4.find("zhao"))
```

# 字符统计：输入任意字符串，统计其中字母、数字、空格的总个数
"""
1. 获取输入的字符串
2. 遍历字符串 获取每一个字符
3.判断字符对应的类型：字母 数字 空格
4、累计 统计数量
5.打印结果
"""
n1=0
n2=0
n3=0

# for i in range(0,len(s)):
    # s[i]
for c in s:
    # c对应的字符
    if c.isalpha():
        # 字母
        n1=n1+1
    elif c.isdigit(): # 数字
        n2+=1
    elif c.isspace(): # 空格
        n3+=1

print(f"{s}字符串的统计结果：字母:{n1}个，数字:{n2}个,空格:{n3}个")

# 3.容器数据类型

容器数据类型：管理多个数据，这一类的数据类型

Python语言的容器数据类型主要有4种：

1.元组 tuple

2.列表list

3.集合set

4.字典dict



## 3.1 元组

元组tuple：不可改变，记录多个不同数据类型的值

定义：变量名=(值,…… )

> 内部元素的值的数据类型可以相同，也可以不同

特点：

不可改变，有序（元素的创建顺序），可以重复

核心：

1.索引访问

2.循环遍历

3.长度 len

示例：

```python
# 元组：1.不可改变 2.定义的符号 ()
# 定义元组类型的变量
t1=(1,11,33,'abc')
# 通过索引获取元素
print("元组的内容：",t1[2])
# 通过for 循环遍历内容
for e in t1:
    print(e)
# 直接打印
print(t1)
# 获取元素个数
print(len(t1))
# 查看对应的数据类型
print(type(t1))
```

## 3.2 列表

列表list：记录多个值的类型，有序（保证添加的顺序），可重复，可以改变（新增、修改、删除、查询-获取）

语法格式： 变量名=[值,……]

特点：

有序、可变、可重复

核心：

1.索引

2.长度 len

3.新增 append （追加）

4.删除 pop(索引) 不传索引删除最后一个元素

5.排序 sort 默认从小到大，如果需要从大到小需要设置 reverse=True

6.查看元素对应的索引 index

7.统计出现的次数 count

8.清空列表 clear

9.操作列表的函数，列表的内容为数字类型：最大值max 最小值min 求和sum

示例：列表基础操作

```python
# 列表 有序 可变 可重复
l1=[11,22,True]
# 访问元素 索引
print(l1[1])
# 类型
print(type(l1))
# 新增
l1.append(33)
# 修改
l1[0]=101
# 删除 pop 不传递参数 表示删除最后一个，传递参数-对应要删除的索引
l1.pop(2)
# 遍历 获取元素
for l in l1:
    print(l)
# 输出整个列表
print(l1)
print(len(l1))
```

列表的小练习：

```python
# 实现一个用户输入新增列表的元素,列表长度大于6就结束输入
print("---新增内容到列表--")

    s=input("请输入新增的元素：")

列表的常用函数或操作：

```
# 列表的常用函数
l=[1,33,4,22,5,4,70]
#  最大值 数字类型
print(max(l))
# 最小值
print(min(l))
# 求和
print(sum(l))
# 排序 sort 排序 ，参数：reverse 是否从大到小，默认为false 从小到大
l.sort(reverse=True)
print("排序：",l)
# index 获取对应元素的索引
print("查询索引：",l.index(22))
# count 统计元素出现的次数
print("次数：",l.count(4))
# clear 清空
l.clear()
print(l)
```
