# 第一章：概率论基础与高斯分布

## 0. 符号约定

- 观测集 X：N 个样本，每个 p 维特征
- 单个样本 x_i = (x_i1, x_i2, ..., x_ip)
- 每个观测由分布 p(x|θ) 生成

---

## 1. 频率派 vs 贝叶斯派

这是机器学习最底层的两个哲学分歧。你选哪一派，决定了你怎么建模、怎么求解。

### 1.1 频率派（Frequentist）

**核心信念：参数 θ 是固定不变的常数，只是我们不知道它而已。**

方法：**极大似然估计 MLE**（Maximum Likelihood Estimation）

```
已知：一堆数据 X
目标：找一个 θ，使得"在这些参数下，出现这组数据的概率最大"

θ_MLE = argmax log p(X|θ)
      = argmax Σ log p(x_i|θ)          ← 取 log 把连乘变连加
```

**用到了什么：** 最优化理论（求导、梯度下降）。

**代表算法：** 线性回归、逻辑回归、SVM（频率派为主）。

### 1.2 贝叶斯派（Bayesian）

**核心信念：θ 不是常数，θ 本身也有一个分布。**

方法：**最大后验估计 MAP**（Maximum A Posteriori）

```
已知：一堆数据 X + 我对 θ 的先验信念 p(θ)
目标：结合数据和先验，更新我对 θ 的认知

p(θ|X) = p(X|θ) × p(θ) / p(X)         ← 贝叶斯公式
        = 似然 × 先验 / 证据

后验 ∝ 似然 × 先验                       ← 分母和 θ 无关，常省略
```

**用到了什么：** 积分（求后验分布、做预测都要积）。MCMC 采样是常用工具。

**代表算法：** 朴素贝叶斯、高斯过程、概率图模型。

### 1.3 对比

| | 频率派 | 贝叶斯派 |
|------|--------|----------|
| θ 是什么 | 固定常数 | 随机变量，有分布 |
| 求解方法 | MLE（最大化似然） | MAP（最大化后验） |
| 数学工具 | 最优化（求导） | 积分（采样） |
| 预测方式 | 代入 θ 直接算 | 对所有可能的 θ 加权平均 |
| 一句话 | "最好的参数是哪个" | "参数可能是多少，每种可能多大概率" |

### 1.4 贝叶斯预测

贝叶斯预测不是"选一个最好的 θ 去预测"，而是**对所有可能的 θ 做加权平均**：

```
p(新数据|旧数据) = ∫ p(新数据|θ) × p(θ|旧数据) dθ
                      ↑ 模型       ↑ 后验（θ 的当前认知）
```

频率派：选一个 θ，赌它是对的。

贝叶斯派：考虑所有 θ，按可能性加权。

## 2. 高斯分布（正态分布）

为什么高斯分布这么重要？三个原因：
1. **中心极限定理**：大量独立随机变量之和趋近高斯分布（自然界最常见）
2. **数学性质好**：条件分布、边缘分布、线性变换后还是高斯
3. **只有两个参数**：均值 μ（在哪）和方差 σ²（有多散）

### 2.1 一维高斯：MLE 推导

概率密度函数（PDF）：

```
p(x|μ,σ²) = (1/√(2π)σ) × exp(-(x-μ)²/(2σ²))
```

**问题：** 给一堆数据 x1, x2, ..., xN，用 MLE 估计 μ 和 σ。

**求 μ：** 对数似然对 μ 求导 = 0

```
→ μ_MLE = (1/N) Σ x_i          ← 就是样本均值！
```

**求 σ：** 把 μ_MLE 代入，再对 σ 求导 = 0

```
→ σ²_MLE = (1/N) Σ (x_i - μ)²   ← 样本方差，但分母是 N
```

### 2.2 重点：为什么 σ²_ML 是有偏的？

这是经典考点。σ²_MLE 的分母是 N，但我们用 N-1。

**推导：**

σ²_MLE 对所有可能数据集的期望 ≠ 真实的 σ²：

```
E[σ²_MLE] = (N-1)/N × σ² ≠ σ²
```

**原因：** 计算 σ² 时用了 μ_MLE（也是从同一批数据估出来的），损失了一个自由度。

**修正：** 用 N-1 代替 N

```
σ̂² = (1/(N-1)) Σ (x_i - μ)²    ← 无偏估计
```

**直观理解：** 你用同一批数据既估计了 μ 又估计了 σ，数据被"用了两次"，所以要减 1 补偿。

> 这和 `np.std(ddof=1)` 是一个道理，ddof=1 就是 N-1。

---

### 2.3 多维高斯分布

```
p(x|μ,Σ) = 1/[(2π)^(p/2) |Σ|^(1/2)] × exp(-½(x-μ)ᵀ Σ⁻¹ (x-μ))
```

| 符号 | 含义 | 维度 |
|------|------|------|
| x | 样本向量 | p×1 |
| μ | 均值向量（中心在哪） | p×1 |
| Σ | 协方差矩阵（形状/方向） | p×p |
| (x-μ)ᵀΣ⁻¹(x-μ) | 马氏距离（到中心的标准化距离） | 标量 |

**马氏距离 vs 欧氏距离：**

- 欧氏距离：只考虑多远，不管方向
- 马氏距离：考虑了各维度的方差和相关，是"标准化后的欧氏距离"

**协方差矩阵 Σ 的特征分解：**

Σ = U Λ Uᵀ，其中：
- U 的列是特征向量（决定了椭圆的方向）
- Λ 的对角线是特征值（决定了椭圆的轴长）

→ 高斯分布的等高线是**椭圆**，方向由特征向量决定，轴长由特征值的平方根决定。

### 2.4 两个实际问题

**问题 1：参数太多**

Σ 有 p(p+1)/2 个自由参数。p=1000 → 约 50 万个参数，根本估不出来。

解决方案：
- 假设 Σ 是对角矩阵（各维度独立）→ Factor Analysis
- 假设 Σ = σ²I（各向同性，球形的）→ 概率 PCA

**问题 2：单峰**

单个高斯只有一个峰（一个 μ）。数据有多个簇怎么办？

解决方案：**高斯混合模型 GMM**，用多个高斯加权组合。

### 2.5 多维高斯的核心定理

> **定理（线性变换）：** 如果 x ~ N(μ, Σ)，则 y = Ax + b ~ N(Aμ + b, AΣAᵀ)

这是整个多维高斯运算的基石，所有推导都基于它。

**证明（不严谨版）：**
- E[y] = E[Ax + b] = A E[x] + b = Aμ + b
- Var[y] = Var[Ax + b] = Var[Ax] = A Var[x] Aᵀ = AΣAᵀ

---

### 2.6 边缘分布与条件分布（均还是高斯！）

把 x 切成两块：

```
x = [x_a]     μ = [μ_a]     Σ = [Σ_aa  Σ_ab]
    [x_b]         [μ_b]         [Σ_ba  Σ_bb]
```

**边缘分布（只看一部分）：**

```
x_a ~ N(μ_a, Σ_aa)              ← 直接取对应块，最简单
x_b ~ N(μ_b, Σ_bb)
```

**条件分布（已知一部分，另一部分是多少？）——这是机器学习预测的核心：**

```
x_b | x_a ~ N(μ_{b·a} + Σ_ba Σ_aa⁻¹ x_a ,  Σ_{bb·a})
```

其中 Σ_{bb·a} = Σ_bb - Σ_ba Σ_aa⁻¹ Σ_ab，叫 **Schur 补（Schur Complement）**。

**通俗理解条件分布的均值：**

```
E[x_b | x_a] = μ_b + Σ_ba Σ_aa⁻¹ (x_a - μ_a)
                ↑     ↑                ↑
              先验   修正系数          x_a 偏离均值的程度
              
= "我以为 x_b 是 μ_b，但看到 x_a 后，根据它们的关系修正一下"
```

**条件分布的方差 Σ_{bb·a}：**

```
知道 x_a 的信息后，x_b 的不确定度降低了：
新方差 = 原方差 - 由 x_a 解释掉的那部分
```

---

### 2.7 线性高斯模型（最重要的应用）

**问题设定：**

```
p(x) = N(μ, Λ⁻¹)                  ← x 的先验
p(y|x) = N(Ax + b, L⁻¹)           ← 给定 x，y 的条件分布
问：p(y) 和 p(x|y) 是什么？
```

**求解思路：** 令 y = Ax + b + ε，ε ~ N(0, L⁻¹)

**结果 1 — p(y)（边缘分布）：**

```
y ~ N(Aμ + b, L⁻¹ + AΛ⁻¹Aᵀ)
     ↑         ↑        ↑
    均值    观测噪声   x的不确定性传播到y
```

**结果 2 — p(x|y)（逆推，贝叶斯推断核心）：**

```
E[x|y] = μ + Λ⁻¹Aᵀ (L⁻¹ + AΛ⁻¹Aᵀ)⁻¹ (y - Aμ - b)
          ↑   ↑                           ↑
        先验 卡尔曼增益                  预测误差

Var[x|y] = Λ⁻¹ - Λ⁻¹Aᵀ (L⁻¹ + AΛ⁻¹Aᵀ)⁻¹ AΛ⁻¹
           ↑                        ↑
        先验不确定性        看到 y 后减少的不确定性
```

**这个模型 = 所有贝叶斯线性回归 / 卡尔曼滤波 / 高斯过程回归的数学基础。**

## 3. 本章公式速查

| 概念 | 公式 | 关键点 |
|------|------|--------|
| MLE（频率派） | θ = argmax Σ log p(x_i\|θ) | 选最可能产生数据的 θ |
| MAP（贝叶斯派） | θ = argmax p(X\|θ)×p(θ) | 结合先验信念 |
| 一维高斯 μ_MLE | (1/N) Σ x_i | 样本均值 |
| 一维高斯 σ²（有偏） | (1/N) Σ (x_i-μ)² | MLE 估计，偏小 |
| 一维高斯 σ²（无偏） | (1/(N-1)) Σ (x_i-μ)² | 修正后 |
| 线性变换定理 | Ax+b ~ N(Aμ+b, AΣAᵀ) | 高斯 × 线性 = 高斯 |
| 边缘分布 | x_a ~ N(μ_a, Σ_aa) | 直接取子块 |
| 条件分布均值 | μ_a + Σ_ab Σ_bb⁻¹ (x_b-μ_b) | 用已知修正未知 |
| 条件分布方差 | Σ_aa - Σ_ab Σ_bb⁻¹ Σ_ba | 已知后不确定度减小 |
| 线性高斯 p(y) | N(Aμ+b, L⁻¹+AΛ⁻¹Aᵀ) | 边缘仍是高斯 |
| 线性高斯 p(x\|y) | 见 2.7 | 贝叶斯推断的数学基础 |

## 4. 这一章在 ML 中的位置

```
频率派 MLE  →  线性回归、逻辑回归、SVM
贝叶斯派 MAP → 朴素贝叶斯、贝叶斯网络
多维高斯条件分布  →  高斯过程、卡尔曼滤波、贝叶斯优化
高斯混合 GMM  →  聚类、生成模型
```

**一句话：** 这章在讲"不确定性的数学语言"。频率派把不确定性归结为数据噪声，贝叶斯派把不确定性归结为参数本身的分布。高斯分布是整个 ML 的通用语言——因为它数学性质太好了。
