## 1.2 类和对象

类：属性和函数的封装，模板

对象：根据类创建的实例（拥有类中定义的属性和函数）

类的定义的语法格式：

```
class 类名:
	# 构造函数 创建对象的时候会调用的函数 内部：1.定义类的属性 2.完成属性的赋值
	def __init__(self,参数名,……):
		# 定义属性并赋值
		self.属性名=参数名
		……
	def 函数名(self,参数名,……):
		实现特定功能的代码块
	
```

对象创建的语法格式：

对象名=类名(值,……)

调用属性

对象名.属性名

对属性赋值

对象名.属性名=值

调用函数

对象名.函数名(值,……)

> 构造函数中定义的参数对应的值

示例代码：

```python
# 类
class Person:
```

## 1.3 封装

面向对象有3大特征：1.封装 2.继承 3.多态

封装的作用：隐藏内部的实现细节，暴漏公共的函数

封装的实现：

1.属性

​	在类中有3种属性：

​	1.公共属性 在哪都可以访问 

​	示例：slef.属性名

​	2.受保护的属性  本类和子类

​	示例：self._属性名

​	3.私有属性 本类

​	示例：self.__属性名

​	

2.私有或受保护的属性的操作函数 

​	get_xxx 获取属性的内容

​	set_xxx  设置属性的内容

3.类中的函数也可以设置为访问受限的

​	1.公共函数 在哪都可调用

​	def 函数名(self,参数名,……):

​		xxx

​	2.受保护的函数 本类和子类中

​	def _函数名(self,参数名,……):

​		xxx

​	3.私有函数 本类

​	def __函数名(self,参数名,……):

​		xxx

示例代码：

```python
class Dog:
    def __init__(self,name,age,type):
        # 公共属性 在哪都可访问
        self.name=name
        # 受保护的属性 本类和子类
        self._age=age
        # 私有属性 本类
        self.__type=type

    # 私有属性 外接无法访问使用，所以一般需要2个函数
    def get_type(self):
        return self.__type
    def set_type(self,type):
        self.__type=type

# 对象
dog1=Dog("旺财",5,"中华田园犬")
print("名称：",dog1.name)
print("年龄：",dog1._age) # 不建议外界访问受保护的属性
print("品种：",dog1.get_type())
dog1.set_type("哈巴狗")
dog1.name="花花"
print(dog1.name)
print(dog1.get_type())
```



```python
# 访问受限的函数
class Phone:
    def __init__(self,model,brand,price):
        self.model=model
        self.brand=brand
        self.price=price
    # 公共函数
    def show(self):
        # 调用私有函数
        if self.__limit_brand():
            # 调用受保护的函数
            print(f"手机{self.model},{self.brand},{self._js_price()}元")
        else:
            print("不是小米手机")
    # 受保护的函数 本类或子类
    def _js_price(self):
        return self.price/100.0
    # 私有的函数 本类
    def __limit_brand(self):
        if self.brand=="小米":
            return True
        else:
            return False
# 对象
p1=Phone("小米16","小米",6909)
p1.show()
p2=Phone("华为Meta8","华为",9999)
p2.show()
```

## 1.4 继承

继承：子承父业

子类继承父类，就可以拥有父类中的非私有的属性和非私有的函数

语法格式：

```
# 子类
class 类名(父类的类名):
	def __init__(self,参数名,……)
		# 调用父类的构造函数
		super().__init__(参数值,……)
		self.属性名=参数值
```

> 继承的是什么：
>
> 1.属性 公共和受保护的，私有属性不会被继承
>
> 2.函数 公共和受保护的，私有函数不会被继承

继承的作用：精简代码，如果多个类拥有公共的属性或函数，那么应该把共同的属性和函数封装成父类

> Python语言允许多继承，一个类可以有多个父类

示例代码：

```python
# 定义动物类
class Animal:
    def __init__(self,name,age,sex):
        self.name=name
        self.age=age
        self.sex=sex
    def eat(self,food):
        print(f"{self.name}喜欢吃{food}")

class Cat(Animal):
    def __init__(self,name,age,sex,type):
        # 调用父类的构造函数
        super().__init__(name,age,sex)
        self.type=type
class Dog(Animal):
    def __init__(self,name,age,sex,price):
        super().__init__(name,age,sex)
        self.price=price

# 对象
# 子类对象
c1=Cat("小花",3,"母","加菲猫")
print(c1.name)
print(c1.age)
print(c1.sex)
print(c1.type)
c1.eat("鱼")
# 子类
d1=Dog("小菜",6,"公",8999)
print(d1.name)
d1.eat("骨头")
```

## 1.5 多态

多态：一个东西，具备多种形态

多态的体现：

1.父类对象和子类对象的转型

​	1.向上转型

​		子类对象自动转换为父类对象

​	2.向下转型

​		父类对象强制转换为子类对象，请务必使用：isinstance判断是否为True,True才可强制

示例：

```python
# 多态 多种形态
# 父类
class Car:
    def __init__(self,brand,price):
        self.brand=brand
        self.price=price
    # 函数
    def sale(self,zk):
        print(self.brand,"打折处理，目前折扣：",zk,"优惠费用：",self.price-self.price*zk)

# 子类
class BydCar(Car):
    def __init__(self,brand,price,tea):
        super().__init__(brand,price)
        self.tea=tea

# 定义函数
def show(car):
    print(type(car))
    print(car.brand,"价格：",car.price)

# 创建对象 子类对象
bc=BydCar("比亚迪",100000,"性价比")
bc.sale(0.8)
# 创建父类对象
c=Car("长城",999999)
c.sale(0.8)
show(c)
c=bc
show(bc)
# 子类对象和自动转换为父类对象 向上转型 自动进行
print(isinstance(c,Car))
print(isinstance(c,BydCar))
```


类中都有str函数，只是默认没有任何输出，默认返回的是对象的地址，想要做其他的输出，就需要重写函数的内容

> str函数重写的要求：1.没有参数 2.返回值必须为字符串
>
> 外接可以直接打印对象即为调用__str函数

示例代码：

```python

class Bar:
    def __init__(self,name,age):
        self.name=name
        self.age=age
    # __str__ 打印对象信息，必须返回值 字符串类型
    def __str__(self):
        print("对象的内容：")
        print(f"名称={self.name}，年龄={self.age}")
        return f"{self.name}-{self.age}"

b=Bar("测试",112)

# 调用 __str__函数
print(b)
```

Python提供了异常处理，用户捕获异常并做出相应操作

异常处理主要有以下关键字：

1.try 内部代码如果出现异常就会结束后续的操作，同时匹配后续的except

2.except 对应的异常类 内部记录对应的异常信息

3.else try内部没有出现异常就会执行else内部的内容

4.finally 最终的，try无论是否有异常，都会执行finally

5.raise 抛出异常（一般都是抛出自定义的异常）



## 2.1 try-except

捕获异常使用try和except

语法格式：

```
try:
	代码块，可能会出现异常的
except 异常类 as 别名:
	通过别名获取异常的具体信息
```

示例代码：

```python
# 异常捕获处理
try:
    num=int(input("请输入一个数字"))
    print(num+"测试")
except Exception as e: # 直接使用父类 子类对象自动转换为父类对象（向上转型）
    print(e)
# except (TypeError,ValueError) as e: # 多个异常类 
#     print(e)
# except ValueError as v:
#     print("亲，出错了：",v)
# except TypeError as t:
#     print("亲，请检查输入的内容：",t)
print("程序运行成功")

```



## 2.2 else

else 就是可以和try搭配使用

try内部的内容如果没有异常，就执行else内部的代码

示例：

```python
# try except else
list1=[11,22]
try: # 可以获取异常
    index=int(input("请输入访问索引："))
    print(list1[index])
except IndexError as e: # 出现了匹配的异常
    print("索引越界：",e)
else: # try内部没有异常就会执行
    print(list1)
```

## 2.3 finally

finally 无论如何都会执行的

```python

finally: # 是否发生异常都会执行
    print("反正我执行")
```

# 3.总结

面向对象相关：消除重复的代码

​	类：共同的属性和函数的封装体，模板

​	对象：通过类创建的实例，拥有类中的属性和函数，通过对象名.属性名或对象名.函数名()

类的语法格式：

```
class 类名:
	# 构造函数
	def __init__(self,参数,……):
		# 定义属性
		self.属性名=参数
		……
	# 定义函数
	def 函数名(self,参数,……):
		实现功能的代码块
	
```

对象的语法格式：

```
对象名=类名(参数值,……)
对象名.属性名
对象名.函数名(参数值,……)
```

封装：属性封装

可以把属性的访问权限进行设置

私有属性或受保护的属性需要提供get和set函数

受保护的属性或函数：_xx

私有的属性或函数：__xx

继承：子承父业

子类可以拥有父类中非私有的属性和函数

语法格式：

```
class 子类名(父类名):

	def __init__(self,参数,……):
		super().__init__(参数值,……)
		# 父类名.__init__(self,参数值,……)
		属性
	……
```

多态：具备多种形态

1.对象转型

​	向上转型：子类对象可以自动转换为父类对象

​	向下转型：父类对象强制转换为子类对象（务必使用 isinstance验证是否可以转换）

> 想向下转型必须先向上转型

2.函数的重写

​	在子类中，可以把父类的函数进行重写，函数都一样，只是内部的实现不同

直接打印对象名就是调用__str函数，默认返回的内容地址

可以重写__str函数实现指定的内容返回

Python语言提供的异常处理

1.try 内部代码如果出现异常就会结束后续的操作，同时匹配后续的except

2.except 对应的异常类 内部记录对应的异常信息

3.else try内部没有出现异常就会执行else内部的内容

4.finally 最终的，try无论是否有异常，都会执行finally

5.raise 抛出异常（一般都是抛出自定义的异常）
