### 1.文档分割

> 文档都是md格式，所以分割方式需要更改

第一版：

separators=["\n\n","\n","<",">","。"," "]

问题：

> - **失 Markdown 专属结构符**：Markdown 文档的核心在于其标题层级（如 `#`, `##`）和列表等结构。当前的 separators 没有包含这些标记，导致切割器无法识别章节边界，可能会将不同标题下的内容强行合并，或者在标题与正文之间产生不合理的切割。
> - **< 和 > 作为分隔符风险较高**：这两个符号在 Markdown 中通常用于引用块（`>`）或 HTML 标签（如 `<br>`）。如果将它们作为普通分隔符，极易破坏 HTML 标签的完整性，或者将引用块的内容切得过于细碎，导致检索时上下文丢失。
> - **中文标点不够全面**：仅使用句号 `。` 作为句子级兜底可能不够，中文语境下的感叹号 `！`、问号 `？` 以及分号 `；` 同样是重要的语义边界。

第二版：

为了更好地适配 Markdown 格式，建议将 separators 调整为“由粗到细”的层级结构，优先保留文档的章节和段落完整性：

separators = [
r"\n#{1,6}\s",      # 1. 优先按 Markdown 标题层级（H1-H6）切割
r"\n\d+(?:\.\d+)*\s", # 2. 按数字编号标题（如 1. / 2.3.）切割
"\n\n",             # 3. 按双换行（段落）切割
"\n",               # 4. 按单换行切割
"。", "！", "？", "；", # 5. 按中文主要句末标点切割
" ",                # 6. 按空格切割
""                  # 7. 兜底：按字符级硬切
]

### 3.混合检索

Hybrid Search=向量（相似性）搜索+关键字搜索

> 混合检索需要使用专门混合检索模型

第一版搜索：

向量的相似性搜索

```
result=chroma.similarity_search(query=question,k=5)
```

第二版引入混合检索

> 向量检索 + 关键词检索（最经典的混合检索）
>
> 这是目前企业级 RAG 系统中最基础且应用最广泛的混合检索策略。
>
> - **向量检索（语义搜索）**：使用 **Embedding 模型**（如 text-embedding-3-large、BGE、instructor-xl 等）将文本转化为高维向量，通过计算余弦相似度来检索。它的优势在于能理解自然语言、同义词和模糊概念，适合处理“为什么”、“怎么做”等语义类问题。
> - **关键词检索（词汇检索）**：通常使用 **BM25 算法**。它依赖反向索引进行精确的词元匹配，在处理产品代码、SKU、法律条款、错误代码等精确匹配场景时表现卓越。
> - **融合机制**：两路检索并行后，通常使用 **RRF（Reciprocal Rank Fusion，倒数排名融合）算法** 将两路结果进行分数重新计算和排序，最后再交由 **Reranker（重排序模型）** 进行精排，从而得到最优的上下文。

召回率提升相当有效，成本低

### 4.重排序

在初始检索阶段，系统通常会优先保证速度，因此返回的结果可能包含大量相关度参差不齐的候选文档（例如返回 50-100 个片段）。由于向量检索主要依赖数学上的相似度分数，更高的相似度并不总是意味着内容最能直接回答用户的问题。重排序模型通过计算用户问题与每一个候选文档之间的深层语义相关性分数，对结果进行重新打分和排序，将最相关的结果排在前面，从而显著提升最终生成答案的质量。

在 RAG（检索增强生成）系统中，重排序（Rerank）是提升检索精度的核心环节。如果说初始的向量检索或混合检索是“粗召回”，那么重排序就是“精排”。

重排序的模型：

qwen3-rerank：问答、语义

示例代码：

```python


# ==================== 重排序（qwen3-rerank via 阿里云百炼）====================
def rerank_documents(query: str, documents: list[str], top_n: int = TOP_K_RERANK) -> list[str]:
    """使用阿里云百炼 qwen3-rerank 模型对候选文档重排序，返回 Top-N 最相关文档。"""
    if not documents:
        return []
    if len(documents) <= top_n:
        return documents

    url = "https://dashscope.aliyuncs.com/api/v1/services/rerank/text-rerank/text-rerank"
    headers = {
        "Authorization": f"Bearer {BAILIAN_API_KEY}",
        "Content-Type": "application/json"
    }
    payload = {
        "model": "gte-rerank",
        "input": {
            "query": query,
            "documents": documents
        },
        "parameters": {
            "top_n": top_n,
            "return_documents": True
        }
    }
    try:
        resp = requests.post(url, headers=headers, json=payload, timeout=30)
        if resp.status_code == 200:
            data = resp.json()
            results = [item["document"]["text"] for item in data.get("output", {}).get("results", [])]
            print(f"[重排序] {len(documents)} 篇 -> Top-{len(results)}")
            return results
        else:
            print(f"[重排序失败] HTTP {resp.status_code}: {resp.text}")
            return documents[:top_n]  # 降级：取前 N 篇
    except Exception as e:
        print(f"[重排序异常] {e}，取前 {top_n} 篇")
        return documents[:top_n]
```

            "top_n": top_n,
            "return_documents": True
        }
    }
    try:
        resp = requests.post(url, headers=headers, json=payload, timeout=30)
        if resp.status_code == 200:
            data = resp.json()
            results = [item["document"]["text"] for item in data.get("output", {}).get("results", [])]
            print(f"[重排序] {len(documents)} 篇 -> Top-{len(results)}")
            return results
        else:
            print(f"[重排序失败] HTTP {resp.status_code}: {resp.text}")
            return documents[:top_n]  # 降级：取前 N 篇
    except Exception as e:
        print(f"[重排序异常] {e}，取前 {top_n} 篇")
        return documents[:top_n]
# ==================== 工具函数 ====================
@tool
def search_rag(question: str) -> str:
    """查询 RAG 知识库获取 IT 技术知识。
    适用场景：LangChain、RAG、Agent、Prompt 工程、向量数据库、Embedding 等技术问题。
    
    Args:
        question: 用户要查询的问题
    
    Returns:
        格式化后的相关文档内容字符串
    """
    # 1. 查询重写：口语 → 检索关键词
    rewritten = rewrite_query(question)

    # 2. 混合检索（向量 + BM25）
    hybrid = load_rag()
    raw_results = hybrid.invoke(query=rewritten)
    print("混合检索结果：", len(raw_results))

    if not raw_results:
        return "未查询到相关数据"

    # 3. 提取文档文本
    contents = [r.page_content for r in raw_results]

    # 4. 重排序（qwen3-rerank）
    reranked = rerank_documents(query=question, documents=contents, top_n=TOP_K_RERANK)

    # 5. 格式化返回
    parts = [f"[文档{i+1}] {text}" for i, text in enumerate(reranked)]
    return "\n\n".join(parts)
